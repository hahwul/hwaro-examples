## Conversion Funnel Analysis

Tracking the user journey from initial page view to completed purchase over the last 30 days.

### Funnel Stages

| Stage | Users | Drop-off |
|-------|-------|----------|
| Landing Page | 84,291 | -- |
| Product View | 41,302 | 51.0% |
| Add to Cart | 12,845 | 68.9% |
| Checkout Start | 5,218 | 59.4% |
| Purchase Complete | 2,731 | 47.7% |

### Overall Conversion Rate: 3.24%

This represents a 0.8 percentage point improvement over last month (2.44%).

### Recommendations

1. **Reduce landing-to-product drop-off** -- The 51% drop-off at the first stage suggests the landing page could better communicate value proposition
2. **Cart abandonment** -- Implement exit-intent recovery emails for the 59.4% who start checkout but don't complete
3. **Mobile checkout** -- Mobile conversion is 40% lower than desktop; prioritize mobile checkout UX improvements
