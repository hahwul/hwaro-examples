This theme was built to demonstrate how far we can push CSS and SVG filters to recreate a classical art technique on the web.

Instead of solid borders, this layout uses repeating background dots to create visual structure. Images are filtered using an SVG `feTurbulence` and `feDisplacementMap` combined with a dot mask to simulate the pointillist painting style.
