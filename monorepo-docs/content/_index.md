Welcome to the **Monorepo Docs** example site. This site demonstrates a sophisticated documentation structure for large-scale monorepo projects.

## Key Features

- **Package Switcher**: Easily navigate between different packages in your monorepo.
- **Package-Aware Sidebar**: The sidebar automatically updates to show relevant documentation for the selected package.
- **Dependency Graphs**: Visualize how your packages interact with each other.
- **Cross-Reference Linking**: Effortlessly link between packages.

Select a package from the top menu or explore the [General Guide](/guide) to get started.
