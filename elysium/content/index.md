## Discover Elegance

Elysium is a beautifully crafted template designed to put your content first. With its careful attention to typography and space, it provides a serene reading experience.

This template is perfect for creatives, writers, and anyone looking to present their work without distractions.

### Getting Started

1. Edit `content/index.md` to change this page.
2. Add new `.md` files in `content/` to create new pages.
3. Update `static/css/style.css` to tweak colors and typography.
4. Run `hwaro build` to generate your elegant site.
