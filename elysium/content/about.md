## The Philosophy

Elysium was built with a simple premise: content deserves to be read in a beautiful environment.

We stripped away unnecessary borders, gradients, and complex layouts to focus on what truly matters: the words on the page.

### Features

- Dark mode by default for reduced eye strain
- Elegant typographic scale combining Inter and Playfair Display
- Fully responsive design
- SEO-friendly structural layout
- Lightweight footprint

Feel free to make it your own by modifying the CSS variables in the styles.
