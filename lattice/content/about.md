# About Lattice

Lattice is a documentation resource for graph database modeling and query patterns. It covers schema design principles, Cypher query language fundamentals, and advanced traversal techniques for building connected data applications.

This documentation is intended for developers, data architects, and database administrators working with property graph databases. Whether you are migrating from a relational model or designing a greenfield graph application, Lattice provides the reference material you need.

## Topics Covered

- **Data Modeling** -- Node labels, relationship types, property schemas, and constraint definitions
- **Query Language** -- Cypher syntax, pattern matching, aggregations, and write operations
- **Performance** -- Indexing strategies, query planning, and optimization techniques
- **Use Cases** -- Social networks, recommendation engines, knowledge graphs, and fraud detection patterns
