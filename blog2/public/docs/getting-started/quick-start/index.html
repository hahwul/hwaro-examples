Create a Project

acme init my-project
cd my-project

This generates the following structure:

my-project/
├── acme.toml
├── src/
│   └── main.ac
└── tests/
    └── main_test.ac

Run the Project

acme run

Your application is now running at http://localhost:4000.

Build for Production

acme build --release

The optimized binary is output to dist/.
