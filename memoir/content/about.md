## Memoir

Memoir is a Hwaro theme designed for long-form writing. It draws inspiration from the physical experience of reading a hardcover novel: generous margins, careful typography, and a pace that invites the reader to slow down.

### Design Principles

**Typography first.** Every decision starts with readability. The pairing of Cormorant Garamond for headings and EB Garamond for body text creates a classic feel rooted in centuries of typographic tradition. Line height is set generously at 1.9, and the measure is kept at a comfortable 38rem.

**Chapter structure.** Content is organized into chapters with clear numbering, epigraphs, and in-page navigation. This mirrors the structure of a printed book, giving readers a familiar sense of progression.

**Quiet ornamentation.** Drop caps, section dividers, and roman numeral counters in the table of contents add personality without distraction. The warm color palette and subtle textures keep the reading experience calm and focused.

### Features

- Elegant serif font pairing (Cormorant Garamond and EB Garamond)
- Drop cap on the first paragraph of each chapter
- Epigraph support with author attribution
- In-chapter table of contents with roman numeral counters
- Previous/next chapter navigation
- Footnote styling
- Wide margins and generous line spacing optimized for long reading sessions
- Responsive layout that preserves readability on all screen sizes
- Bookmark-style chapter listing with numbering

### Built With

This theme is built with [Hwaro](https://hwaro.hahwul.com), a fast static site generator written in Crystal. No JavaScript frameworks are required. The entire theme is built with HTML templates, CSS, and Markdown content.
