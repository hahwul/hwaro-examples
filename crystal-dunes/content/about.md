Crystal Dunes is an example site designed to showcase an elegant, glassmorphic layout mixed with a warm, desert-inspired color palette.

It demonstrates the flexibility of Hwaro's templating system by completely refactoring the default scaffold to use template inheritance and custom CSS.

```crystal
# Example code block styling

  puts "Welcome to the dunes, #{name}!"
end
```
