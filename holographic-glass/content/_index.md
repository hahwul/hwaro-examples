Welcome to the **Holographic Glass** example. This theme demonstrates advanced CSS techniques to create a futuristic, immersive interface using translucent surfaces and vivid, glowing colors.

### Key Features

*   **Glassmorphism:** Employs `backdrop-filter: blur()` to create elegant, frosted glass panels.
*   **Animated Holographics:** Features continuously floating, softly glowing orbs in the background.
*   **Layered Aesthetics:** Uses varied opacities, inset shadows, and carefully controlled text colors to maintain contrast within a dark theme.
*   **Dynamic Interactions:** Hovering over the main glass panel reveals a subtle, sweeping reflection effect.

### Typography & Code

This theme uses the `Outfit` font for a modern, geometric look. Code blocks are styled to fit seamlessly into the dark environment:

```crystal
# Example Crystal Code
def hello_hologram(name : String)
  puts "Welcome to the future, #{name}!"
end

hello_hologram("Hwaro User")
```
