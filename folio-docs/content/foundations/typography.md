## Font Families

| Token | Value | Usage |
|-------|-------|-------|
| `--font-sans` | Inter, system-ui | Body text, UI elements |
| `--font-mono` | JetBrains Mono | Code, technical content |

## Type Scale

| Name | Size | Line Height | Weight | Usage |
|------|------|-------------|--------|-------|
| Display | 36px | 1.2 | 700 | Page titles |
| Heading 1 | 30px | 1.3 | 700 | Section titles |
| Heading 2 | 24px | 1.35 | 600 | Sub-section titles |
| Heading 3 | 20px | 1.4 | 600 | Group titles |
| Body Large | 18px | 1.6 | 400 | Lead paragraphs |
| Body | 16px | 1.6 | 400 | Default body text |
| Body Small | 14px | 1.5 | 400 | Captions, labels |
| Caption | 12px | 1.4 | 500 | Badges, metadata |

## Guidelines

- Limit heading levels to three (H1, H2, H3) in most contexts
- Use Body Large for introductory paragraphs only
- Monospace font is reserved for code and technical identifiers
- Maintain consistent line length: 60-80 characters per line for body text
