# About

This is a personal blog built with an aurora-inspired aesthetic -- deep, rich colors drawn from the northern lights against a dark sky, paired with refined typography and generous whitespace.

The site is powered by [Hwaro](https://github.com/hahwul/hwaro), a fast static site generator written in Crystal. It features tags, pagination, RSS feeds, syntax highlighting, and responsive design.

## The Theme

Aurora is a design language defined by the colors of the northern sky. Key characteristics include:

- **Deep dark backgrounds** evoking the night sky
- **Teal and violet accents** inspired by aurora borealis
- **Elegant serif headings** paired with clean sans-serif body text
- **Generous spacing** for a calm, unhurried reading experience
