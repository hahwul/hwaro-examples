Typography is not decoration. It is the structural framework of written communication. Every choice -- typeface, size, weight, spacing -- is a decision about hierarchy, rhythm, and meaning.

## The Hierarchy of Attention

A page without typographic hierarchy is a wall. The eye has nowhere to land, no path to follow. Good typography creates an implicit reading order:

1. The headline arrests attention
2. The subhead provides context
3. The body text delivers substance
4. The metadata recedes without disappearing

This hierarchy is not aesthetic preference. It is functional architecture.

## Serif vs. Sans-Serif

The debate is tired. Both have their uses. What matters is consistency and intention:

- **Serif faces** carry weight and tradition. They signal permanence. In body text, their letter shapes create a baseline rhythm that guides the eye.
- **Sans-serif faces** project clarity and modernity. In interfaces and short-form content, their simplicity reduces friction.

The mistake is using one where the other belongs, or mixing them without purpose.

## Spacing as Meaning

The space between elements communicates relationship. Tight spacing groups things together. Wide spacing creates separation. This is not about "looking clean" -- it is about encoding meaning into layout.

Consider two list items with 4px between them versus 24px. The first says "these belong together." The second says "these are distinct." No words were needed.

## The Rule of Restraint

Use fewer typefaces. Use fewer weights. Use fewer sizes. The constraint is not limitation -- it is discipline. A page set in a single typeface at three sizes will always outperform a page set in five typefaces at seven sizes.

Restraint is what separates typography from font selection.
