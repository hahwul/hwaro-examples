# Ethereal Glass & Glowing Tides

Welcome to **Opal Tide**, an example site demonstrating the capabilities of Hwaro with a modern, dark glassmorphism aesthetic.

The design features animated, glowing orbs that create a dynamic backdrop, combined with frosted glass panels that blur the background and provide an elegant container for your content.

## Features

* **Glassmorphism UI:** Translucent panels with blur effects.
* **Animated Backgrounds:** Fluid, glowing CSS orbs in pastel cyan, magenta, and purple.
* **Typography:** Clean and modern look using *Outfit* for headings and *Inter* for body text.

> "Simplicity is the ultimate sophistication."
