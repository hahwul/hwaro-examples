# Snowfall: Winter-Themed Blog

A serene, calm, and pure winter-themed seasonal blog template for Hwaro.

## Design

- **Theme**: Pure white (#ffffff), ice blue (#e3f2fd), and silver (#bdc3c7) color palette.
- **Vibe**: Peaceful, cold but cozy, clean, and minimal.
- **Features**:
  - CSS-only falling snow particle animation.
  - Frost-textured card borders with subtle shadows.
  - Sticky header with elegant typography.
  - **No Gradients & No Emojis**: Strictly uses solid colors and clean icons for a professional look.
- **Typography**: Inter / System-ui sans-serif.

## Build

```bash
hwaro build -i snowfall
```

The output will be generated in `snowfall/public/`.
