Cherry blossoms have inspired artists for centuries. In web design, the sakura aesthetic brings a unique blend of elegance and boldness that few other motifs can match.

## Why Sakura Works in Design

The cherry blossom carries layers of meaning — beauty, transience, renewal. These qualities translate powerfully to visual design:

- **Contrast** — Soft petal shapes against bold color fields create instant visual tension
- **Motion** — Falling petals naturally suggest animation and flow
- **Palette** — The pink-to-white gradient range offers incredible versatility
- **Emotion** — Sakura evokes warmth, nostalgia, and wonder simultaneously

## Techniques Used Here

This theme uses several CSS techniques to channel the cherry blossom aesthetic:

### Gradient Storms

```css
background: linear-gradient(135deg, #ff1493, #c71585, #8b008b);
```

Multi-stop gradients create the illusion of petals layered in wind, shifting from hot pink through magenta to deep violet.

### Glow Effects

```css
text-shadow: 0 0 20px rgba(255, 20, 147, 0.6),
             0 0 40px rgba(255, 105, 180, 0.3);
```

Layered `text-shadow` creates a warm bloom glow, as if text is illuminated by blossom-filtered light.

### Floating Particles

CSS `@keyframes` animations drive the falling petal effects, using `translateY` and `rotate` to simulate wind-scattered movement across the viewport.

## The Result

When combined, these techniques create an immersive atmosphere that transforms a simple blog into a visual experience — one that captures the explosive beauty of a sakura storm.
