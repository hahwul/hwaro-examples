## About Brocade

Welcome to Brocade — a theme inspired by the art of luxurious textile weaving. Like the intricate raised patterns of gold and silver threads woven into rich silk, this site celebrates craftsmanship, beauty, and attention to detail.

## Design Philosophy

Brocade textiles have been prized across cultures for centuries. From the silk workshops of Byzantine Constantinople to the looms of Renaissance Florence, brocade represented the pinnacle of decorative art. This theme captures that spirit:

- **Rich color palette** — Deep burgundy, antique gold, and warm ivory
- **Metallic accents** — Shimmering highlights that evoke gold thread
- **Textured layers** — Subtle patterns that create visual depth
- **Bold typography** — Elegant serif fonts paired with modern readability

## Built With

This site is built with [Hwaro](https://github.com/hahwul/hwaro), a fast and lightweight static site generator written in Crystal. Pure CSS — no JavaScript frameworks required for the visual effects.
