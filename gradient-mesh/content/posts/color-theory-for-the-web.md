Color is the most immediate element of any design. Before a visitor reads a single word, they've already absorbed your color palette and formed an emotional impression.

## The Color Wheel Revisited

While most designers know the basics — complementary, analogous, triadic — the web introduces unique considerations:

- **Screen luminance** changes how colors interact
- **Dark backgrounds** shift perceived saturation
- **Accessibility** demands sufficient contrast ratios

## Building a Vibrant Palette

For the Gradient Mesh theme, we chose a palette that spans the warm-to-cool spectrum:

| Color | Role | Hex |
|-------|------|-----|
| Fuchsia | Primary accent | `#ff6b9d` |
| Violet | Secondary accent | `#c084fc` |
| Cyan | Tertiary accent | `#06b6d4` |
| Amber | Warm highlight | `#f59e0b` |

These four colors, when blended through mesh gradients, create a rich tapestry of intermediate hues — magentas, teals, lavenders — without ever feeling chaotic.

## Practical Tips

1. Use **CSS custom properties** for your palette — easy to adjust globally
2. Test on multiple monitors — color rendering varies significantly
3. Always check contrast with tools like the [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/)

Great color is intentional, not accidental.
