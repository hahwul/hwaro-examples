## About This Site

Welcome to **Gradient Mesh** — a glamorous, eye-catching blog theme built with [Hwaro](https://github.com/hahwul/hwaro), a fast static site generator written in Crystal.

## Design Philosophy

This theme embraces bold visual spectacle through fluid gradient mesh backgrounds with organic, multi-color transitions. Every page features complex color blending that creates a sense of depth and motion.

### Features

- Fluid multi-color gradient mesh backgrounds with animated transitions
- Bold, vibrant color palette with rich visual layers
- Glassmorphism card surfaces over dynamic backdrops
- Smooth CSS animations for depth and engagement
- Clean, readable typography with generous spacing
- Fully responsive design for all screen sizes
- Syntax highlighting with Atom One Dark theme

## Technology

Pure CSS animations and gradients — no JavaScript required for the visual effects. The mesh backgrounds use layered radial gradients with CSS keyframe animations to create organic, flowing color transitions.
