# Lighthouse Example

Lighthouse is an elegant, curated navigation guide and resource directory theme for Hwaro. It focuses on clean typography, structured data, and efficient resource discovery.

## Features
- **Clean Aesthetic:** Light theme with zero gradients and emojis for a professional, distraction-free experience.
- **Resource Directory:** Resources grouped by categories.
- **Search Integration:** Built-in search for resources, tags, and categories.
- **Alphabetical Index:** Easy navigation through curated resources.
- **Metadata Support:** Badges for "Recommended" status and ratings.

## Getting Started
1. Install Hwaro.
2. Clone this repository.
3. Run `hwaro serve` in this directory.

## Configuration
Customize your directory in `config.toml` and add your own categories and resources in the `content/` folder.
