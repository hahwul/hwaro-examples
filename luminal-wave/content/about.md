Luminal Wave is an advanced, highly stylized example site for Hwaro. It demonstrates the power of CSS variables and modern layout techniques.

### Core Features

*   **Fluid Gradients**: Animated radial gradients create a dynamic background.
*   **Glassmorphism**: Translucent panels with blur effects provide a sense of depth.
*   **Neon Accents**: Bright cyan and magenta colors contrast against the deep dark background.
*   **Responsive Typography**: Uses modern fonts like Space Grotesk and Inter for a clean look.

This design aims to provide a visually striking experience while maintaining readability.
