# Scaffold

A command-line tool for bootstrapping and scaffolding new projects from structured templates.

## What is Scaffold?

Scaffold generates project directory trees from predefined or custom templates. It reads a template definition, prompts for variables, and writes the output to disk -- giving every new project a consistent starting point.

## Key Capabilities

- **Template-driven generation** -- Define reusable project structures as template packs
- **Variable interpolation** -- Inject project name, author, license, and custom values into file contents and paths
- **CLI-first workflow** -- All operations run from the terminal with scriptable flags
- **Directory tree output** -- Produces clean, predictable directory layouts every time
- **Custom templates** -- Author and share your own template packs
