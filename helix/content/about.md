## About Helix

Helix is a biotech-inspired theme for Hwaro, designed for teams and individuals working at the intersection of biology and technology. The visual language draws from the structure of DNA -- the double helix, base pair complementarity, and the elegant machinery of molecular biology.

## Design Principles

This theme is built around core ideas from the life sciences:

- **Base pair pairing** -- Content is structured in complementary pairs, echoing the A-T and G-C bonds that hold the double helix together
- **Clean lab aesthetic** -- A light, precise color palette inspired by modern research facilities
- **3D CSS helix structure** -- Decorative elements built with pure CSS transforms, no images required
- **Cell division transitions** -- Subtle animations that evoke the rhythm of mitosis
- **Scientific typography** -- Space Grotesk and IBM Plex Mono for clarity and precision

## Technology

This site is built with [Hwaro](https://github.com/hahwul/hwaro), a fast static site generator written in Crystal. The theme uses no JavaScript frameworks -- just clean HTML, CSS custom properties, and vanilla JS for search.
