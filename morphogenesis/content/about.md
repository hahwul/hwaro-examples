# About Morphogenesis
Morphogenesis is a dark theme inspired by biological pattern formation and generative growth algorithms.
## Features
- Responsive layout for all devices
- Custom color palette and typography
- Clean post cards with hover effects
- Tag-based content organization
- RSS feed support
## Getting Started
```bash
hwaro init my-site --scaffold https://github.com/hahwul/hwaro-examples/tree/main/morphogenesis
```
