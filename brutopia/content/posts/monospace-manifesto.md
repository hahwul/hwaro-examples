Variable-width fonts optimize for reading speed. Monospace fonts optimize for structural clarity. In a brutalist context, the choice is obvious.

<!-- more -->

## Equal Width, Equal Weight

Every character in a monospace typeface occupies the same horizontal space. This means:

1. **Alignment is inherent** -- columns form naturally
2. **Counting is visual** -- character length is proportional to space
3. **Code and prose share a grid** -- no typographic hierarchy is imposed externally

## The Terminal Heritage

Monospace type carries the memory of the terminal. Of machines that printed one character at a time, each in its fixed-width cell. There is no pretense in a terminal. What you type is what exists.

```
BRUTOPIA
========
concrete. code. clarity.
```

## Against Typographic Hierarchy

Traditional web design uses font size, weight, and family to establish hierarchy. Brutopia uses only three signals:

- **Uppercase** for headings
- **Color** for emphasis
- **Spacing** for structure

The typeface remains constant. The grid does the work.
