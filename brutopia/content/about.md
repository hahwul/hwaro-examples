# About

Brutopia is a dark brutalist theme for the web. It draws from the exposed concrete and deliberate austerity of brutalist architecture, translated into a digital context with monospace type and a strict color discipline.

Built with [Hwaro](https://github.com/hahwul/hwaro), a fast static site generator written in Crystal.

## Design Principles

- Monospace type throughout -- uniform, mechanical, honest
- Dark ground with warm concrete tones
- Red as the sole accent -- structural, not decorative
- Borders define hierarchy, not shadows or depth effects
- Every pixel earns its place or it is removed
