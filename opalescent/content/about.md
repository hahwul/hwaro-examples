## About Opalescent

Opalescent is a glamorous, eye-catching blog theme for [Hwaro](https://github.com/hahwul/hwaro) inspired by the mesmerizing play of colors seen in opal gemstones.

### Design Philosophy

This theme embraces bold visual spectacle through:

- **Iridescent color-shifting** — CSS gradients and animations create a shimmering, opal-like effect across the interface
- **Soft rainbow highlights** — Subtle prismatic accents add depth without overwhelming readability
- **Ethereal glow** — Carefully layered shadows and light effects create a dreamy, floating atmosphere
- **Dynamic transitions** — Smooth animations bring the interface to life

### Technical Details

Built entirely with CSS custom properties and modern CSS features:

- CSS `@keyframes` animations for the signature color-shift effect
- `backdrop-filter` for frosted glass surfaces
- CSS `conic-gradient` and `linear-gradient` for rainbow borders
- Custom properties for easy theme customization
