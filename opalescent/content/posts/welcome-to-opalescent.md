Welcome to **Opalescent**, a theme that captures the ethereal beauty of opal gemstones in a digital format.

<!-- more -->

## The Inspiration

Opals are unique among gemstones because they don't just reflect light — they *diffract* it. A single stone can contain the entire visible spectrum, shifting and shimmering as you change your viewing angle.

This theme attempts to capture that same sense of wonder through carefully crafted CSS animations and gradient effects.

## What Makes It Special

Every element in this theme has been designed with visual richness in mind:

- **Animated gradient backgrounds** that shift through the opal spectrum
- **Glassmorphism cards** with frosted surfaces and rainbow borders
- **Glowing accents** that pulse subtly to create a living, breathing interface
- **Smooth transitions** on every interactive element

## Getting Started

To use this theme with Hwaro, simply clone the example and start customizing:

```bash
hwaro init my-site --scaffold https://github.com/hahwul/hwaro-examples/tree/main/opalescent
hwaro serve --open
```

The color palette is fully customizable through CSS custom properties — change the look entirely by updating a few variables.
