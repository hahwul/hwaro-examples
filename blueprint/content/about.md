## Project Overview

Blueprint is a documentation system designed for engineering teams that produce technical specifications, RFCs, and design documents. Every document follows a structured lifecycle from draft to approval.

## Document Standards

All specifications in this repository adhere to the following conventions:

- **Unique Identifier**: Each spec is assigned a `SPEC-XXX` identifier
- **Status Tracking**: Documents move through DRAFT, REVIEW, and APPROVED states
- **Revision Control**: Each revision is numbered and dated
- **Author Attribution**: Every document has a designated owner

## Tooling

This site is generated using [Hwaro](https://github.com/hahwul/hwaro), a fast and lightweight static site generator written in Crystal. The blueprint theme emphasizes readability of technical content including code blocks, specification tables, and structured metadata.
