# About Astro Pulse

**Astro Pulse** is an example theme built for the Hwaro static site generator.

It demonstrates how to effectively combine deep dark themes with modern CSS features like `backdrop-filter` for glassmorphism effects, alongside CSS animations to create a "pulsing" neon nebula effect in the background.

## Features

- **Responsive Design:** Looks great on mobile, tablet, and desktop.
- **Glassmorphism UI:** Clean, frosted-glass containers.
- **Animated Backgrounds:** Subtle, moving radial gradients to simulate a living cosmos.
- **Performance:** Built with lightweight HTML/CSS, perfectly suited for Hwaro's fast rendering.
