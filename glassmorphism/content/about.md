# About

Welcome. This is a personal blog built with a frosted glass aesthetic -- translucent panels, soft blur effects, and clean typography against a dark backdrop.

The site is powered by [Hwaro](https://github.com/hahwul/hwaro), a fast static site generator written in Crystal. It features tags, pagination, RSS feeds, syntax highlighting, and responsive design.

## The Theme

Glassmorphism is a design language defined by frosted-glass surfaces. Key characteristics include:

- **Translucent backgrounds** with backdrop blur filters
- **Subtle borders** that catch light at the edges
- **Soft shadows** for depth without harshness
- **Restrained color palette** that lets the glass effect take center stage
