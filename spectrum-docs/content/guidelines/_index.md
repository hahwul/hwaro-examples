# Accessibility Compliance Levels

The Web Content Accessibility Guidelines (WCAG) are organized into three levels of conformance. Each level includes several success criteria that must be met to achieve that level of accessibility.

### Level A: Essential
The most basic level of accessibility. If these requirements are not met, many users with disabilities will find it impossible to use the website.

### Level AA: Standard
The standard level of accessibility. This is the target level for most professional and government websites. If these requirements are not met, some users with disabilities will find it difficult to use the website.

### Level AAA: Enhanced
The highest level of accessibility. Meeting these criteria makes the website accessible to the widest possible range of users with disabilities.
