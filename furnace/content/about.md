# About Furnace

Furnace is a curated collection of performance engineering documentation covering benchmarking methodologies, profiling techniques, and optimization strategies for production software systems.

## Purpose

This site serves as a reference for engineers working on performance-critical applications. It consolidates practical knowledge around CPU profiling, memory analysis, load testing, caching strategies, and query optimization into a single, searchable resource.

## Scope

Furnace covers the full performance engineering lifecycle: establishing baselines, identifying bottlenecks, applying optimizations, and validating results. Examples use real-world tooling including pprof, perf, wrk, and standard database profiling utilities.

## Contributing

Contributions are welcome. All benchmark results should be reproducible, and optimization recommendations should include before/after measurements demonstrating measurable improvement.
