## About Foxhole

Foxhole is a security-focused writeup blog documenting CTF challenge solutions, vulnerability research, and penetration testing techniques. Each writeup follows a structured approach: reconnaissance, analysis, exploitation, and flag capture.

### Categories

- **Web** -- SQL injection, XSS, SSRF, authentication bypass, and web application exploitation
- **Pwn** -- Binary exploitation, buffer overflows, heap attacks, and shellcoding
- **Reversing** -- Static and dynamic analysis, deobfuscation, and protocol reverse engineering
- **Crypto** -- Cryptographic attacks, weak implementations, and mathematical exploits
- **Forensics** -- Memory analysis, network captures, disk forensics, and steganography

### Tools

Commonly used tools across writeups:

| Category | Tools |
|----------|-------|
| Web | Burp Suite, sqlmap, ffuf, httpx |
| Pwn | pwntools, GDB/GEF, ROPgadget, one_gadget |
| Reversing | Ghidra, IDA, radare2, Binary Ninja |
| Forensics | Wireshark, Volatility, binwalk, foremost |
