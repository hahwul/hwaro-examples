## About This Site

Welcome to **Liquid Chrome** — a theme that captures the mesmerizing beauty of molten metal and mirror-like surfaces. Built with [Hwaro](https://github.com/hahwul/hwaro), a fast static site generator written in Crystal.

## Design Philosophy

This theme draws inspiration from liquid chrome 3D art — flowing metallic surfaces that catch light in unexpected ways, creating organic reflections and distortions.

### Visual Features

- Animated gradient backgrounds with chrome-like flow
- Metallic text effects with reflective highlights
- Glassmorphic cards with frosted blur
- Smooth hover transitions and glow effects
- Iridescent accent colors that shift across the spectrum
- Dark palette with high-contrast metallic accents
- Responsive design for all screen sizes

## Technology

Pure CSS animations and effects — no JavaScript frameworks required for the visual design. The theme uses CSS custom properties, `backdrop-filter`, `linear-gradient` animations, and `text-shadow` layering to achieve its metallic look.
