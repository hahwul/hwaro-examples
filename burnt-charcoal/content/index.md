# Deep. Textured. Black.

Welcome to the Burnt Charcoal theme. This theme explores the depths of textured blacks with occasional glowing embers at the borders.

## The Ember Aesthetic

The glowing ember aesthetic is achieved through carefully layered box-shadows and border elements, avoiding the use of CSS gradients. It provides a stark contrast against the dark background.

<div class="charcoal-block">
This is a custom charcoal block. Notice the glowing ember border on the right.
</div>

### Features

- **Textured Backgrounds**: Utilizing SVG noise to create a rough, ashy feel.
- **Glowing Accents**: Sharp contrasts with ember-like colors (oranges, reds, yellows).
- **Strictly No Gradients**: All glows are created with shadows.

> "In the ashes of what once was, glowing embers remain."
> — *The Charcoal Manifesto*

```css
/* Example of the ember effect */
.ember-accent {
    background-color: var(--ember-orange);
    box-shadow: 0 0 8px var(--ember-orange), 0 0 15px var(--ember-red);
}
```
