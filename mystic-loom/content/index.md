# Weaving the Void

Welcome to the **Mystic Loom**. This template showcases a deep space, dark-themed aesthetic characterized by translucent glassmorphism panels, glowing woven ambient light animations, and sophisticated typography using Space Grotesk and Inter.

## Discover the Threads

Here within the loom, content is woven from markdown into beautiful static pages.

*   Edit `content/index.md` to alter the fabric of this home page.
*   Weave new threads by creating `.md` files in the `content/` directory.
*   Explore the [About](/about/) section.
*   Run `hwaro serve` to witness the loom in action with live reload.
