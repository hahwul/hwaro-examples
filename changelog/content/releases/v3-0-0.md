## Added

- New plugin architecture supporting third-party extensions
- `hwaro plugin install` command for managing plugins
- Hot module replacement for template development
- Built-in image optimization pipeline with WebP/AVIF output
- `config.validate` option for strict configuration checking

## Changed

- **Breaking:** Template engine upgraded to Crinja v2 with new filter syntax
- **Breaking:** Configuration format migrated from YAML to TOML
- Improved build performance by 3x through parallel content processing
- Redesigned CLI output with structured logging

## Removed

- **Breaking:** Dropped support for legacy `.yml` configuration files
- Removed deprecated `page.params` accessor (use `page.extra` instead)
- Removed built-in jQuery dependency from default templates
