# About this microblog

This is a microblog built with [Hwaro](https://github.com/hahwul/hwaro). Short thoughts, links, and notes that don't need a full blog post.

Posts are sorted by time, newest first. Each entry is a self-contained note. Think of it as a public notebook.

## Colophon

- Static site generator: Hwaro
- Theme: micro
- No JavaScript frameworks, no build tools, just HTML and CSS
