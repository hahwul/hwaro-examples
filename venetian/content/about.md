# About Venetian
Venetian is a rich theme inspired by Renaissance Venice with deep crimsons, gold, and canal blues.
## Features
- Responsive layout for all devices
- Custom color palette and typography
- Clean post cards with hover effects
- Tag-based content organization
- RSS feed support
## Getting Started
```bash
hwaro init my-site --scaffold https://github.com/hahwul/hwaro-examples/tree/main/venetian
```
