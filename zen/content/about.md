## About This Site

Zen is a blog theme built for [Hwaro](https://github.com/hahwul/hwaro), the fast static site generator written in Crystal. Its design draws from Japanese minimalism and the philosophy of wabi-sabi.

## Design Principles

**Ma (negative space)** -- The layout uses generous whitespace to let content breathe. Every element has room around it, creating a sense of calm and focus.

**Wabi-sabi** -- Rather than chasing perfection, this theme embraces simplicity and restraint. There are no flashy animations, no heavy frameworks, just text and space.

**Tategaki (vertical text)** -- A subtle vertical text element on the side pays homage to traditional Japanese typography, where text flows from top to bottom, right to left.

**Restrained palette** -- Colors are drawn from natural materials: warm paper, soft ink, muted earth tones. Nothing demands attention; everything serves the reading experience.

## Features

- Serif typography with Noto Serif JP
- Generous line-height and reading measure
- Thin, delicate dividers
- Responsive single-column layout
- RSS feed, sitemap, and taxonomy support
- Syntax highlighting for code blocks
