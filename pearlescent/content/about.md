# About Pearlescent
Pearlescent is a soft, luxurious theme inspired by the iridescent shimmer of pearls.
## Features
- Responsive layout for all devices
- Custom color palette and typography
- Clean post cards with hover effects
- Tag-based content organization
- RSS feed support
## Getting Started
```bash
hwaro init my-site --scaffold https://github.com/hahwul/hwaro-examples/tree/main/pearlescent
```
