## About

Tale is a classic blog theme for Hwaro, inspired by the original [Tale Jekyll theme](https://github.com/chesterhow/tale) by Chester How. It embraces a timeless aesthetic with serif body text, clean navigation, and a focus on readability.

### Features

- Serif typography for comfortable reading
- Clean, date-based post listing
- Minimal and distraction-free layout
- No JavaScript required
- CSS under 300 lines
- Responsive design
- Syntax highlighting for code blocks
- RSS/Atom feed support

### Setup

1. Clone or download the files into your Hwaro project
2. Run `hwaro serve` from the terminal
3. Open your browser to `http://localhost:3000/`

Enjoy writing!
