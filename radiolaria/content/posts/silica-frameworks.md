Just as silica provides the rigid skeleton for radiolarians, CSS variables and semantic HTML provide the framework for this theme.

### The CSS Variables

```css
:root {
    --bg-color: #0d131a;
    --text-color: #d1e2eb;
    --accent: #ff3366;
    --border: rgba(209, 226, 235, 0.15);
}
```

By defining these at the root level, we ensure a unified structural identity. Modifying the `--accent` color instantly propagates a new energy throughout the entire system, much like an injection of dye into a capillary network.

Every element is encased in a logical container. There is no ambiguity.
