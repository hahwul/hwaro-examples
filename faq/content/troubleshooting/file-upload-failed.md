Common reasons for file upload failures:

- **File too large** -- Free plans allow up to 25 MB per file; Pro allows 100 MB; Enterprise allows 500 MB
- **Unsupported format** -- Check that your file type is supported (most common formats are accepted)
- **Network interruption** -- Unstable connections can cause uploads to fail midway. Try a wired connection.
- **Storage quota exceeded** -- Check your storage usage in **Settings** then **Billing**

If the issue continues, try uploading from a different browser or in an incognito/private window.
