## About

Emerald is a minimal theme created for Hwaro. The main purpose of Emerald is to provide a clear theme for those who want a blog ready to use, focused on the content and mobile-first.

The original Emerald theme was designed by [Jacopo Rabolini](https://github.com/KingFelix/emerald) for Jekyll. This version has been adapted for the [Hwaro](https://github.com/hahwul/hwaro) static site generator.

### Features

- Minimal and clean design
- Mobile-first responsive layout
- Slide-in navigation menu
- Customizable color scheme
- Syntax highlighting for code blocks
- RSS/Atom feed support

### Setup

1. Clone or download the files into your Hwaro project
2. Run `hwaro serve` from the terminal
3. Open your browser to `http://localhost:3000/`

Enjoy!
