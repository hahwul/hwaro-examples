# Notation Guide

This page defines the standard notation used throughout the Formulary. Consistent notation helps avoid ambiguity when reading formulas across different sections.

## General Symbols

| Symbol | Meaning |
|--------|---------|
| $\mathbb{R}$ | Set of real numbers |
| $\mathbb{Z}$ | Set of integers |
| $\mathbb{C}$ | Set of complex numbers |
| $\mathbb{N}$ | Set of natural numbers |
| $\mathbb{Q}$ | Set of rational numbers |
| $\emptyset$ | Empty set |
| $\in$ | Element of |
| $\subset$ | Proper subset |
| $\subseteq$ | Subset or equal |
| $\forall$ | For all |
| $\exists$ | There exists |

## Greek Letters

| Letter | Name | Common Use |
|--------|------|------------|
| $\alpha, \beta, \gamma$ | alpha, beta, gamma | Angles, parameters |
| $\delta, \epsilon$ | delta, epsilon | Small quantities, limits |
| $\lambda$ | lambda | Eigenvalues, rates |
| $\mu, \sigma$ | mu, sigma | Mean, standard deviation |
| $\pi$ | pi | Circle constant ($\approx 3.14159$) |
| $\phi, \psi$ | phi, psi | Angles, wave functions |
| $\omega$ | omega | Angular frequency |
| $\Sigma$ | capital sigma | Summation |
| $\Pi$ | capital pi | Product |

## Operations

| Notation | Meaning |
|----------|---------|
| $f'(x)$ or $\frac{df}{dx}$ | Derivative of $f$ with respect to $x$ |
| $\frac{\partial f}{\partial x}$ | Partial derivative |
| $\int f(x)\,dx$ | Indefinite integral |
| $\int_a^b f(x)\,dx$ | Definite integral from $a$ to $b$ |
| $\sum_{i=1}^{n} a_i$ | Sum from $i=1$ to $n$ |
| $\prod_{i=1}^{n} a_i$ | Product from $i=1$ to $n$ |
| $\lim_{x \to a} f(x)$ | Limit as $x$ approaches $a$ |
| $\nabla f$ | Gradient of $f$ |

## Matrix and Vector Notation

Vectors are denoted by bold lowercase letters ($\mathbf{v}$) or with an arrow ($\vec{v}$). Matrices use bold uppercase letters ($\mathbf{A}$). The transpose of a matrix is written $\mathbf{A}^T$, and its inverse as $\mathbf{A}^{-1}$.

The determinant of $\mathbf{A}$ is written $\det(\mathbf{A})$ or $|\mathbf{A}|$. The identity matrix of size $n$ is $\mathbf{I}_n$.
