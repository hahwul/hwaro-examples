## Quick Reference

Some of the most frequently referenced formulas:

| Formula | Name |
|---------|------|
| $a^2 + b^2 = c^2$ | Pythagorean theorem |
| $e^{i\pi} + 1 = 0$ | Euler's identity |
| $\frac{d}{dx}[f(g(x))] = f'(g(x)) \cdot g'(x)$ | Chain rule |
| $\int_a^b f(x)\,dx = F(b) - F(a)$ | Fundamental theorem of calculus |
| $P(A \mid B) = \frac{P(B \mid A)\,P(A)}{P(B)}$ | Bayes' theorem |
