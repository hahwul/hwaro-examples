## About Op Art

Op Art — short for Optical Art — is an art movement that emerged in the 1960s, using geometric patterns and stark contrasts to create the illusion of movement, vibration, and depth on flat surfaces. Artists like Bridget Riley and Victor Vasarely pioneered this style, proving that static images could pulse with energy.

This site is a tribute to that tradition, built with [Hwaro](https://github.com/hahwul/hwaro), a fast static site generator written in Crystal.

## Design Principles

- **High contrast** — Black and white as the foundation, with calculated color accents
- **Geometric precision** — Repeating patterns, concentric forms, and mathematical grids
- **Perceived motion** — CSS animations that evoke the illusion of movement Op Art is known for
- **Visual tension** — Interplay between order and optical distortion

## Features

- Pure CSS optical illusion background patterns
- Animated concentric ring hero section
- Gradient text effects with shifting color cycles
- Minimal typography with strong visual hierarchy
- Responsive layout optimized for readability
- Reduced motion support for accessibility
- Dark theme with vibrant accent palette
