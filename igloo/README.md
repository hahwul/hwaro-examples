# Igloo - Nordic Minimalist Theme

Igloo is a Hwaro theme inspired by Scandinavian and Nordic design principles. It features a clean, airy layout with a focus on typography and a cold, professional color palette.

## Features

- **Minimalism**: Extremely generous white space and clean lines.
- **Cold Tones**: Ice Blue and Slate Grey accents on a pure white background.
- **Pure CSS**: No heavy frameworks, just clean and efficient styling.
- **No Distractions**: No gradients, no emojis, just content.

## Installation

1. Clone the repository.
2. Copy the `igloo` directory to your Hwaro project.
3. Update your `config.toml` to point to the new theme (if applicable).

## Usage

Run Hwaro to build your site:

```bash
hwaro build
```
