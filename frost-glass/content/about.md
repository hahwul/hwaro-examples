# About Frost Glass

The Frost Glass theme was created to showcase how Hwaro can be used to build highly visual, modern web experiences. It blends the performance of a static site generator with cutting-edge CSS techniques.

## Design Philosophy

- **Depth:** Creating layers using blur and transparency.
- **Contrast:** Balancing deep blacks with vibrant neons.
- **Motion:** Subtle, ambient animations that bring the page to life without being distracting.
