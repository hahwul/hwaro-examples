# Welcome to Midnight Blossom

Enter a digital sanctuary where nature meets the neon grid. *Midnight Blossom* is a cyber-botanical glassmorphism theme that blends organic aesthetics with futuristic glows.

## The Digital Garden

In the quiet of the night, code blooms like bioluminescent flora.

> "Where the circuitry entwines with the vines, we find a new kind of beauty."

### Features

- **Ethereal Glows**: Ambient pink and purple gradients drifting in the background.
- **Frosted Glassmorphism**: Translucent panels that reveal the colors underneath.
- **Modern Typography**: Pairing `Space Grotesk` for sharp, futuristic headings with `Outfit` for smooth, readable text.
