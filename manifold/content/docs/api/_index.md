This section covers the Manifold API surface. The platform exposes two separate API servers:

- **Admin API** (port 3001) -- Manage tenants, tiers, and platform configuration
- **Tenant API** (port 3000) -- Tenant-scoped operations for application data

Start with [Admin API](/docs/api/admin-api/) for platform management, review [Tenant API](/docs/api/tenant-api/) for application-level endpoints, and see [Authentication](/docs/api/authentication/) for token and session management.
