# About Athena

Athena is more than just a theme; it's a philosophy of design. We believe that content should breathe, and that the container should never overshadow the art it holds.

Created for [Hwaro](https://github.com/hahwul/hwaro), this scaffold provides everything you need to start building a beautiful, statically generated portfolio in minutes.

## Core Principles

1. **Typography First**: Leveraging classic serif typefaces.
2. **Restraint**: Using a muted, sophisticated color palette.
3. **Focus**: Eliminating unnecessary visual noise.
