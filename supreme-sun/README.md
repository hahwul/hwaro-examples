# Supreme Sun

Supreme Sun is an elegant community board theme for [Hwaro](https://github.com/hahwul/hwaro). It focuses on a clean, sophisticated design inspired by classic forum boards but with a modern, high-end feel.

## Key Features

- **Elegant Design**: Refined typography and a soft, warm color palette.
- **Board Style**: Traditional table-based board layout for easy browsing.
- **Categorized Sections**: Pre-configured sections for `Notice`, `Daily`, `Photos`, and `Q&A`.
- **Light Theme**: Focused on a bright and welcoming atmosphere.
- **Constraint-Driven**: No gradients and no emojis for a distraction-free experience.

## Getting Started

1.  Build the site:
    ```bash
    hwaro build
    ```
2.  Serve locally:
    ```bash
    hwaro serve
    ```

## Customization

-   **Colors**: Modify `--primary` and other variables in `static/css/style.css`.
-   **Layout**: Adjust `templates/section.html` to change the board appearance.
-   **Content**: Add new markdown files to the respective directories in `content/`.
