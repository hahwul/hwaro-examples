## About Spectrum

Spectrum is an accessibility and inclusive design theme for [Hwaro](https://github.com/hahwul/hwaro).

This demo site was created to showcase an elegant design that adheres to strict accessibility guidelines.

## Features

- WCAG AAA contrast ratios
- Clear focus indicators
- Skip navigation for screen readers
- Large, legible typography
- Clean, distraction-free layout
