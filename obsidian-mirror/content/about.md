This theme was built as a study in minimal, dark UI design with a focus on CSS glassmorphism and sharp typography.

## Design Philosophy

The goal was to create a UI that feels solid, cold, and razor-sharp. 

*   **Typography**: Relies on `Inter` for general readability and structural integrity, while `JetBrains Mono` provides a technical, precise feel for metadata and code blocks.
*   **Texture**: A very subtle SVG noise filter is applied to the background to break up the flat black and give a slight grainy texture, like unpolished stone.
*   **Interaction**: Hover states are designed to be snappy and distinct, using glowing text shadows to simulate heat beneath the surface.

### System Specifications

| Element | Description |
| :--- | :--- |
| Background | #050505 with SVG Noise |
| Glass Panels | rgba(20, 20, 22, 0.4) with blur(12px) |
| Accents | Subdued Magma Red (#e04a3d) |
| Fonts | Inter, JetBrains Mono |
