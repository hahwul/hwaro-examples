# About the Theme

**Chromatic Nebula** was crafted with the following features:

- Animated, slow-moving CSS multi-color radial gradient background (`linear-gradient` with animation).
- Frosted glass UI containers (`backdrop-filter: blur(16px)` and semi-transparent borders).
- High contrast, sans-serif typography ensuring dark-mode readability.
- Clean layout using Hwaro's Crinja template engine.

## Usage

Simply generate your content as Markdown files inside the `content` directory. Hwaro handles the rest.
