# About Anthology

Anthology is an SDK and library reference documentation site. It provides comprehensive guides, API references, and code examples for integrating with the Anthology platform across Python, JavaScript, and Go.

## Project Goals

- Provide clear, language-specific documentation for every SDK method
- Maintain consistent behavior across all supported languages
- Offer practical code examples for common integration patterns
- Document error handling, pagination, and authentication in detail

## Contributing

Contributions to the SDK and documentation are welcome. Each language SDK is maintained in its own repository and follows the contribution guidelines of that ecosystem.
