The studio has been thinking about interfaces that report more than they request. A dial reports. A button requests. A needle reports. A toggle requests. The best field instruments are made almost entirely of dials and needles, because a field is not the place to ask a user to *make a choice*.

## Working rule

1. If a user has to choose, assume the choice is wrong and reduce the options.
2. If the options cannot be reduced, consider whether the interface should not exist.
3. If the interface must exist, remove one option anyway.

We are considering a set of fully reportive interfaces for the next transmission. Full notes will run in the studio&rsquo;s autumn dispatch.
