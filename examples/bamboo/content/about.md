## About Bamboo

Bamboo is a sustainability-focused blog theme for Hwaro, designed with the environment in mind.

### Design Philosophy

This theme draws inspiration from the natural world. The color palette uses earth tones, olive greens, and terracotta to reflect a deep connection to nature. Every design decision prioritizes clarity, readability, and minimal resource usage.

### Features

- **Natural color palette** -- olive, earth tones, and terracotta
- **Lightweight design** -- minimal CSS, no JavaScript frameworks
- **Clean typography** -- serif body text paired with sans-serif headings
- **Responsive layout** -- works beautifully on all screen sizes
- **Tag-based organization** -- categorize posts by sustainability topics
- **RSS feed** -- subscribe to stay updated

### Built With

This site is powered by [Hwaro](https://hwaro.hahwul.com), a fast static site generator written in Crystal. Static sites consume fewer server resources and produce lower carbon emissions compared to dynamic alternatives.
