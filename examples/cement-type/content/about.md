Cement Type is a writing journal set entirely in display type on raw concrete. The palette is three flat greys, pure black, and a single reserve colour used nowhere on the page. There are no gradients, no drop shadows, no hover tricks, no decorations that pretend to soften the material.

## What we pour

- **Essays** on typography, building materials, and the structural role of words.
- **Slabs**, which are shorter pieces. A slab fits on a single page and carries weight.
- **Ledgers**, which are reference tables for long-running research.

## What we refuse

- Illustrations that do not carry load.
- Colours that do not punctuate a structure.
- Typefaces that suggest softness.

> Every page is a wall. If a word does not hold up the ceiling, it comes down.

Write if you have a subject that can be cast in display type. Otherwise we recommend a different material.
