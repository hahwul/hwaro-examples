Display type, set at sufficient size and weight, carries the structural role once given to ornament. A masthead that fills the page is not a decoration. It is a wall. The reader leans on it to know where the argument begins.

## Weight is not size

A typeface at 120 pixels is large. A typeface at 120 pixels, set in Archivo Black, is *heavy*. The distinction matters. Size tells the eye where to look first. Weight tells the eye *what is holding up the room*. The two are easily confused, and the confusion produces pages that feel bigger than they are built.

## Four paragraphs

1. A page needs one structural element. Two is redundant. Zero is unbuilt.
2. Display type should sit on its own grid. The body below it should not argue.
3. If two headlines compete, one is a subheading, and should be re-cast in the lower weight.
4. Remove ornament until the remaining type carries the load. Then stop.

> A page that reads like a building looks like a building.

We set every slab on this principle. It does not produce pretty pages. It produces pages that do not flex.
