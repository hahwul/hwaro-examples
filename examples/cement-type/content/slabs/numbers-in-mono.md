Numbers, references, coordinates, and grades are labels. Labels belong in monospace. They are not part of the argument. They are the tools the reader uses to find the argument again.

A number set in display type reads as decoration. Decoration lies about importance. A label that lies is no longer a label.

## House rule

- Display type for arguments.
- Body type for paragraphs.
- Monospace for references.

Nothing crosses those lines. The discipline is small. The effect is large.
