Manifesto Ten rules
One material

  1. Set the page in one display typeface. Never two.

  2. Use flat colours. Do not blend. Do not fade. Do not gradient.

  3. Every rule on the page must carry structural meaning.

  4. Whitespace is a beam, not a cushion. Do not pad.

  5. If a decoration is not load-bearing, remove it.

  6. Type sizes step in fifths, not in sevenths.

  7. Borders are either 3 pixels or 10 pixels. Nothing between.

  8. Rotate nothing. Skew nothing. Pretend no softness.

  9. Numbers are labels, not ornaments. Print them in mono.

  10. If a page is finished, remove one more element. Then publish.
