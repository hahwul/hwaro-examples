The ceiling rig above the main stage was checked and tested this morning. Forty bags of confetti sit in the rig ready for the Sunday drop. The test run used a quarter-load to verify the release mechanism; audio-visual synchronisation tests run tomorrow.

A second drop test is scheduled for Saturday afternoon with the crew only.
