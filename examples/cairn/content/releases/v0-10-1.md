A thin stone. Shale is a bugfix release with no new features and no config
changes — safe to flash on top of any v0.10.x board.

<!-- more -->

## Fixed

- **Ghost keys on 1.5u+ stabilized switches.** The scanner's settle window was
  measured from the wrong edge on boards with diode-per-key `ROW2COL` wiring,
  which let a slow-rising signal register twice. The window is now anchored to
  the strobe, not the read.
- **`cairn flash` hang on RP2040.** When the bootloader enumerated slowly, the
  flasher waited on a mount that never appeared. It now polls with a timeout and
  falls back to `--raw` block writes.

  ```bash
  cairn flash --board sweep36 --timeout 15s
  # falls back automatically; force with:
  cairn flash --board sweep36 --raw
  ```

- **Encoder detents double-counted** during fast rotation because the quadrature
  decoder missed a state on the 4-step-per-detent path. The decoder now tracks
  the full Gray-code cycle.

## Upgrade notes

No `board.toml` changes. This release only touches the scanner and the host-side
flashing tool, so a straight reflash is enough. If you were pinning to v0.10.0 to
avoid the encoder regression, this is the release to move to.
