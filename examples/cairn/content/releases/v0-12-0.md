The largest release since the runtime rewrite. Wireless halves now stay in sync
across sleep, the scripting language ships a standard library instead of a pile
of globals, and combos have been rebuilt so they compose with layers instead of
fighting them.

<!-- more -->

## Added

- **`std` scripting library.** Layer scripts no longer reach for magic globals.
  Behaviors are imported explicitly, which makes keymaps readable and lets the
  compiler catch typos at build time instead of at scan time.

  ```lua
  local tap_hold = require("std.behavior").tap_hold
  local leader   = require("std.leader")

  layer("base", function(k)
    k.bind("caps", tap_hold(key.ESC, mod.CTRL, { term_ms = 180 }))
    leader.map(k, { "g", "w" }, macro.text("git worktree "))
  end)
  ```

- **Per-half battery telemetry** surfaced over the link and exposed to scripts
  as `board.peer.battery`, so a status layer can dim its RGB when the peripheral
  half drops below a threshold.
- **`cairn doctor`** — a preflight that lints your keymap, reports unresolved
  bindings, and estimates bytecode size before you flash.
- **Chorded one-shot layers**: `layer.oneshot("sym")` now stacks with mods, so a
  symbol layer plus shift no longer eats the shift.

## Changed

- The **wireless split link** was rewritten around a wake-on-keypress protocol.
  Both halves negotiate a shared epoch on connect, so a keypress on the sleeping
  peripheral no longer drops while the radio spins up. Median wake latency fell
  from 42 ms to 6 ms.
- **Combos are layer-aware.** A combo defined on a layer only fires while that
  layer is active, and combo term is now per-combo rather than global.
- Debounce moved to a per-switch **eager-then-defer** algorithm; chatter-prone
  switches can opt into a longer settle without slowing the rest of the board.
- `board.toml` gains a `[link]` table; the old top-level `split_*` keys are read
  with a deprecation warning through v0.13.

  ```toml
  [link]
  transport = "ble"     # ble | wired
  poll_hz = 200
  wake_on_keypress = true
  ```

## Fixed

- Home-row mods no longer misfire during fast rolls when two mod keys are held
  within the tapping term; the resolver now uses release order, not press order.
- One-shot layers correctly clear after a chorded key rather than sticking until
  the next tap.
- RGB underglow no longer flickers on the peripheral half during link
  renegotiation.
- Fixed a bytecode alignment bug that truncated macros longer than 64 events on
  M0+ targets.

## Upgrade notes

Run `cairn doctor` before flashing. Keymaps using the old global behaviors build
with warnings but still work; migrate to `require("std.behavior")` at your
convenience. The `[link]` change is backward compatible until v0.13.0.
