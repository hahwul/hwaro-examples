Postmark is a weekly broadsheet on the moving parts of the web platform: what shipped to Baseline, what landed behind an experimental flag, and what moved through a standards committee. It goes out every Monday morning and has run continuously since issue No. 1 — the full run is in the [issue archive](/issues/). Nothing here is sponsored, and there is no roundup filler — if a week is quiet, the issue is short.

## How flags get tested

Every "behind a flag" item in an issue is run against a clean browser profile on the channel named in the copy — Canary, Nightly, Technology Preview, or a dev-channel WebView build. We record the exact flag string so a reader can flip it themselves in under a minute. If a flag is renamed or removed before publication, the issue notes it rather than silently updating the text after the fact.

## How Baseline status is read

"Newly Available" means the feature has landed in the current stable release of all Baseline-tracked engines within roughly the last thirty days; "Available" means it has cleared the additional interoperability window Baseline requires before treating a feature as safe to ship without a fallback. Postmark does not editorialize about whether a feature is production-ready — that call depends on your audience's browser mix, not ours.

## Corrections

Errors get a correction in the next issue's closing section, not a silent edit to the original. If a flag string, a spec link, or a status changes between when an issue is written and when a reader opens it days later, that drift is expected and is exactly the kind of thing a dated weekly is supposed to make visible rather than paper over.

## Masthead mark

The stamp beside the lead issue's dateline is a postage-meter cancellation mark — the circular imprint and the wavy killer bars a sorting machine used to cancel a stamp so it couldn't be reused. Postmark borrows the mark as a dateline: once an issue such as [No. 214](/issues/issue-214/) is stamped, the reporting in it is closed, corrections notwithstanding.

## Get in touch

Postmark is written and edited by the same small desk every week. Corrections, flag tips, and spec-room sightings we missed are always welcome — reach the desk through the address printed at the foot of every issue.
