# About Moroccan Zellige
Moroccan Zellige is a vibrant theme inspired by the hand-cut tile mosaics of Moroccan architecture.
## Features
- Responsive layout for all devices
- Custom color palette and typography
- Clean post cards with hover effects
- Tag-based content organization
- RSS feed support
## Getting Started
```bash
hwaro init my-site --scaffold https://github.com/hahwul/hwaro-examples/tree/main/moroccan-zellige
```
