# Welcome to Cyber Crystal Lotus

This is a newly forged dimension of light and glass generated by [Hwaro](https://github.com/hahwul/hwaro). Experience the deep void background and glowing neon pink and cyan gradients.

## Ethereal Beauty

1. Edit `content/index.md` to infuse your own data.
2. Add new `.md` files in `content/` to expand the neural net.
3. Run `hwaro build` to compile the reality.
4. Run `hwaro serve` to stream the preview locally.
