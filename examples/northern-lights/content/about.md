# About Northern Lights
Northern Lights is a dark theme inspired by the aurora borealis with deep navy, green, and purple gradients.
## Features
- Responsive layout for all devices
- Custom color palette and typography
- Clean post cards with hover effects
- Tag-based content organization
- RSS feed support
## Getting Started
```bash
hwaro init my-site --scaffold https://github.com/hahwul/hwaro-examples/tree/main/northern-lights
```
