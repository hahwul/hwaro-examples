## Added

- Multilingual routing with language-specific content variants
- AMP (Accelerated Mobile Pages) generation for content sections
- `default_language` configuration option
- Language switcher data available in templates via `page.translations`
- Per-language feed generation

## Fixed

- Live reload disconnecting after idle timeout on macOS
- Shortcode rendering order incorrect when nested inside blockquotes
- `hwaro doctor` not detecting invalid taxonomy references in front matter
