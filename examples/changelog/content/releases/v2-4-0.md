## Added

- Asset pipeline with bundling, minification, and fingerprinting
- `auto_includes` feature for automatic CSS/JS loading from static directories
- Deployment configuration with multiple target support
- `hwaro deploy` command with dry-run and confirmation options
- `{{ asset(name="main.css") }}` template function for resolved asset paths

## Changed

- Static file handling now supports content-based cache busting
- Build output includes asset manifest for integration with external tools
- Development server proxies WebSocket connections for live reload
