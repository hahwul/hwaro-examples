## Added

- Built-in image processing with `resize_image()` template function
- LQIP (Low-Quality Image Placeholder) generation for progressive loading
- Auto-generated OpenGraph images for posts without a custom `image` field
- PWA manifest and service worker generation
- `hwaro doctor --fix` for auto-correcting common configuration issues

## Deprecated

- `page.params` accessor is deprecated in favor of `page.extra` and will be removed in v3.0
- YAML configuration files (`.yml`) are deprecated; migrate to TOML format
