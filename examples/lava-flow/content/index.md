Welcome to **Lava Flow**, an experimental theme for Hwaro.

The background is generated using CSS gradients, `mix-blend-mode`, filters, and keyframe animations to create slowly shifting, glowing organic shapes that resemble lava in the dark.

### Features
* Organic, morphing blobs via `border-radius` animation.
* Glowing elements with intense orange, red, and yellow gradients.
* Dark, contrast-heavy typography on a translucent, glassmorphic container.

[View the demo posts](/posts/) to see how lists look.
