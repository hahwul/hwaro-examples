# Luminal Glass

Experience the vibrant flow of colors combined with a sleek glassmorphism design. This static site generated by [Hwaro](https://github.com/hahwul/hwaro) demonstrates how easily you can customize templates to create modern, immersive web experiences.

## Features

- **Animated Gradients**: A continuously flowing background that breathes life into the page.
- **Glassmorphism UI**: Semi-transparent panels with backdrop blurs that provide depth.
- **Responsive**: Fully responsive design.
