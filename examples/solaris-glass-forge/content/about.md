# About Solaris Forge

The **Solaris Glass Forge** is an experimental design exploring the intersection of deep space themes and intense solar phenomena, encapsulated within modern web design principles.

### The Vision

We aimed to create an interface that feels both vast and intimate. The dark background represents the infinite void of space, while the glowing, animated orbs simulate the dynamic and unpredictable nature of solar flares and stellar bodies.

The glass panels act as our viewing ports, protecting the user while allowing the light and energy of the background to filter through in a softened, beautiful manner.

### The Technology

This example is built with [Hwaro](https://hwaro.hahwul.com/), demonstrating its capability to handle custom templates and advanced CSS styling effortlessly.

```css
/* Example Glassmorphism */
.glass-panel {
  background: rgba(255, 255, 255, 0.03);
  backdrop-filter: blur(24px);
  border: 1px solid rgba(255, 255, 255, 0.08);
  border-radius: 24px;
}
```
