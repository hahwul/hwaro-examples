Databending is the practice of opening a media file in a tool not designed for it, making changes, and observing the result. Open a photograph in a text editor. Replace a string. Save. The image survives, but altered -- streaked, shifted, recolored by the corruption.

The process is part experimentation, part archaeology. You learn the internal structure of file formats by breaking them. JPEG headers, PNG chunk sequences, BMP pixel arrays -- each responds differently to interference.

Some corruptions produce nothing useful. Others create images that no algorithm was designed to generate.
