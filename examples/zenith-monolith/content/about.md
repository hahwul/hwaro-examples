This project serves as a reference implementation for brutalist structural design.

The layout utilizes CSS Grid to construct rigid, unyielding columns. Typography relies on the Inter font for primary reading and IBM Plex Mono for structural elements and headers, emphasizing the raw data structure over aesthetic embellishment.

Data is parsed. Structure is rendered.
