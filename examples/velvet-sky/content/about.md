# About Velvet Sky

Velvet Sky is a theme built for [Hwaro](https://hwaro.hahwul.com), designed to be elegant without relying on excessive gradients. It uses subtle borders, refined typography, and a dark, muted palette to let the content shine.

## Features

- **Typography-focused:** Uses *Playfair Display* for headings and *Inter* for body text.
- **Minimalist:** No heavy gradients or distracting elements.
- **Responsive:** Adapts cleanly to mobile and desktop screens.
