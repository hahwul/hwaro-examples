Typography in a glamorous theme needs to balance elegance with readability. The wrong font choice can either look too plain against vibrant visuals or compete with them for attention.

<!-- more -->

## The Opalescent Type System

This theme uses a deliberate pairing of two typefaces:

### Display: Sora

**Sora** is a geometric sans-serif with a modern, slightly futuristic feel. Its clean lines and balanced proportions make it perfect for headings that need to command attention without fighting the iridescent effects.

### Body: Inter

**Inter** is designed specifically for screen readability. Its generous x-height and open letterforms ensure comfortable reading even against the rich visual backdrop.

## Hierarchy Through Weight and Space

Rather than relying on size alone, the type hierarchy uses a combination of:

- **Font weight** — Bold for titles, medium for subtitles, regular for body
- **Letter spacing** — Slightly increased tracking on headings for an airy, luxurious feel
- **Line height** — Generous spacing (1.7) for body text to maintain readability
- **Color contrast** — Bright white for primary text, soft lavender for secondary

## Tips for Glamorous Type

1. **Less is more** with decorative fonts — use them only for key headings
2. **Generous whitespace** lets bold visuals breathe
3. **Consistent vertical rhythm** creates order amid visual richness
4. **High contrast ratios** ensure accessibility isn't sacrificed for style
