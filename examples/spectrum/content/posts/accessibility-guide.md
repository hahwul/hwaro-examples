The Spectrum theme is designed with strict accessibility guidelines in mind. Here is an overview of the key features.

## High Contrast Colors

The theme uses a WCAG AAA compliant color scheme, with high contrast text and link colors to ensure maximum readability for users with visual impairments.

## Skip Navigation

A "Skip to main content" link is included at the very top of the page. It is visually hidden but becomes visible when a keyboard user tabs to it, allowing them to bypass repetitive navigation links.

## Clear Focus Indicators

All interactive elements (links, buttons, form fields) feature a distinct focus indicator, improving usability for keyboard-only navigation. The outline provides clear visual feedback on which element has the current focus.

## Distraction-Free Design

The simple, elegant layout removes unnecessary elements like gradients and emojis that can be distracting or cause issues with screen readers.
