# Embrace the Void

Experience a fresh static site generated by [Hwaro](https://github.com/hahwul/hwaro), wrapped in an elegant and minimal **Obsidian Silk Glass** theme.

## Elegance in Shadows

The Obsidian Silk Glass aesthetic features deep, dark backgrounds with subtle, silk-like light gradients, frosted glass elements, and refined typography.

1. Edit `content/index.md` to change this page.
2. Add new `.md` files in `content/` to create new pages.
3. Run `hwaro build` to regenerate the site.
4. Run `hwaro serve` to preview changes locally.

## Features

- **Deep Obsidian Backdrop**: A sophisticated canvas for your content.
- **Silk Glass Containers**: Subtle, translucent frosted glass cards.
- **Refined Typography**: Modern typefaces utilizing *Space Grotesk* and *Inter*.
