Welcome to **Neo Frosted Vortex**, a dark-themed glassmorphism aesthetic featuring a swirling neon vortex background.

This example demonstrates how to create a visually striking layout using CSS backdrop-filters, animated gradients, and blurred abstract shapes to achieve a modern "glass" effect over a deep, glowing void.

### Features

- Dark background (`#020108`) with a subtle radial gradient.
- Animated, rotating, highly-blurred orbs (Cyan, Magenta, Deep Purple) simulating a vortex.
- Frosted glass containers (`backdrop-filter: blur(20px)`) with semi-transparent backgrounds and subtle borders.
- Typography powered by **Space Grotesk** for headings and **Inter** for body text.
