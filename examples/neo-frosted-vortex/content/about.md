This is the **About** page for the Neo Frosted Vortex example.

The glassmorphism aesthetic relies heavily on the CSS `backdrop-filter` property, which allows the animated background elements to beautifully bleed through the UI panels.

### Design Philosophy

1.  **Depth:** Use blurring and opacity to create layers.
2.  **Color:** Employ vibrant neon colors (cyan, magenta, purple) against a very dark background to make the UI pop.
3.  **Motion:** Subtle, continuous animation of the background orbs brings the interface to life without being overly distracting.
