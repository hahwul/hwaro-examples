# About the Nexus

The Apolo Stellar Nexus is a deep-space data station running on the Hwaro static engine. It serves transmissions through frosted-glass panels, drifting orbs, and a typography pairing of Space Grotesk and Inter.

This page exists to show how a regular content page sits beside the cosmic landing template. The same global stylesheet powers it, so headings, lists, and quotes all inherit the muted neutral palette without any extra theme work.

## What you will find here

- A small set of demo pages that lean into the interstellar aesthetic.
- Predictable Liquid templates for index, section, page, and taxonomy views.
- A modest taxonomy split across `tags` and `categories` so search and filtering surfaces stay populated even on a tiny site.

## How to extend it

Drop a new markdown file into `content/`, set the front matter, and rebuild. The section template will pick it up automatically and the navigation card on the home page will update in place. There is nothing dynamic to wire up and nothing to register.

Most edits live in three files: `templates/index.html` for the landing, `templates/page.html` for body pages, and `static/css` for the frosted glass surfaces.
