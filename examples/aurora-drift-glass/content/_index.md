Welcome to **Aurora Drift Glass**. This is a beautiful, dark-themed sample site designed for Hwaro. It features animated background orbs that resemble drifting auroras, combined with elegant glassmorphism UI elements.

### Features

* **Deep Space Aesthetics:** A rich dark background with vibrant, blurred accent colors.
* **Glassmorphism:** Frosted glass panels using `backdrop-filter: blur(16px)` for a modern look.
* **Animated Background:** Slow-drifting aurora orbs that bring the page to life.
* **Modern Typography:** Clean sans-serif fonts using Space Grotesk and Inter.

This theme is perfect for portfolios, landing pages, or elegant blogs that want to make a striking visual impression.

> "In the quiet of the night, the aurora dances across the dark canvas, an ethereal symphony of light."
