# Welcome to Lumina Prism

This is a beautiful, dark-mode static site generated by [Hwaro](https://github.com/hahwul/hwaro) featuring a glassmorphism design with glowing elements.

## Features

- **Glassmorphism**: Translucent panels with background blur.
- **Dark Mode**: Optimized for low-light environments with vibrant neon accents.
- **Responsive**: Adapts gracefully to all screen sizes.

Check out the [About](/about/) page for more details.
