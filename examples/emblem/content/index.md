Ten years scoring interactive systems, not just cues. I build music that listens back — states, RTPCs, and stingers wired straight into a game's own logic, so the track shifts because the player did something, not because a timer fired.

Most engagements start the same way: a director hands me a vertical slice with a linear loop bolted on, and asks why the music never seems to match what's happening. The answer is usually structural, not melodic — the loop needs to become a system.
