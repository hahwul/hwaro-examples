This theme was built as a demonstration of Hwaro's powerful templating and the beauty of modern CSS.

By combining simple HTML structures with advanced CSS features like `backdrop-filter`, `radial-gradient`, and custom animations, we can create an immersive, ethereal experience.

{% alert(type="Note") %}
The background orbs are created entirely with CSS! No images or JavaScript are required to render the smooth, glowing aurora effect.
{% end %}

### The Color Palette
The theme relies on a deep space background contrasted with vibrant neon accents:
- **Cyan**: `#00f0ff`
- **Magenta**: `#ff0055`
- **Purple**: `#8a2be2`

Feel free to customize these CSS variables in `base.html` to create your own unique neon glassmorphism style.
