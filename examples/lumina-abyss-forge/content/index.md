Welcome to the **Lumina Abyss Forge**. This is a dark-themed example site built with [Hwaro](https://github.com/hahwul/hwaro), featuring a deep abyss background, animated lumina glows, and frosted glass containers.

It utilizes modern CSS features like `backdrop-filter` to create an elegant glassmorphism effect, and CSS animations to provide a subtle, breathing background.

### Features
* Deep dark theme with glowing neon accents (Cyan, Purple, Pink)
* Frosted glassmorphism panels
* Animated pulse backgrounds
* Clean, readable typography using `Inter` and `Space Grotesk`

You can explore the different pages or check out the [About](/about/) section.
