The **Neon Abyss Prism** was forged in the digital crucible as an experiment in modern web aesthetics, pushing the boundaries of CSS glassmorphism and background filtering.

### Technical Stack

This project was built using the following technologies:

1.  **Hwaro**: A blazingly fast static site generator written in Rust, utilizing Crinja for powerful templating.
2.  **HTML/CSS**: Hand-crafted CSS properties including `backdrop-filter`, complex `radial-gradient` backgrounds, and intricate `box-shadow` layering.
3.  **Typography**: Curated fonts from Google Fonts: *Space Grotesk* for striking display headers and *Inter* for highly readable body text.

### The Architect

Crafted by a lone voyager seeking beauty in the dark web space. The goal was to create a design that feels both futuristic and elegant, combining cyberpunk neon aesthetics with the clean lines of modern UI.
