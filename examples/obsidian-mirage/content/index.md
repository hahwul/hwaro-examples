# Enter the Obsidian Mirage

Welcome to a fresh static site generated by [Hwaro](https://github.com/hahwul/hwaro), now shrouded in the deep void glassmorphism aesthetic of **Obsidian Mirage**.

## The Illusion of Depth

This theme explores the intersection of deep darkness and radiant light. The interplay of shadows and shimmering cyan accents creates an environment that feels both expansive and mysterious.

{{ alert(type="info", body="This site uses the `backdrop-filter` property to create stunning glass-like panels over a subtle starry abyss.") }}

## Getting Started

1. Edit `content/index.md` to change this page and explore the void.
2. Add new `.md` files in `content/` to expand your universe.
3. Run `hwaro build` to forge the site.
4. Run `hwaro serve` to preview the mirage locally.
