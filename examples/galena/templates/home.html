Distributed cron · jitter · backfills · dead-letter queues

Schedules that fire, catch up, and never drop.

Galena is a distributed job scheduler. It spreads fire times with jitter, replays missed windows as bounded backfills, and parks failed jobs in a dead-letter queue instead of losing them.

Quickstart

5 minutes

Install the daemon, define one job, and watch a leader elect itself. No message broker required to start.

brew install galena
galena serve --state ./galena.db
