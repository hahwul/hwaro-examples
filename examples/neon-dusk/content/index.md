## System Initialization

Welcome to **Neon Dusk**. This layout is designed for projects that need an immediate visual impact, utilizing deep dark backgrounds, bright neon accents, and modern typography.

## Features

- **Cyberpunk Aesthetics:** Bold neon colors, grid-based background accents, and glowing text.
- **Responsive Layout:** Adjusts seamlessly from desktop terminals to mobile datapads.
- **Custom Typography:** Integrated with 'Orbitron' and 'Rajdhani' for that authentic sci-fi feel.

## Quick Start

1. Initialize your project using this setup.
2. Edit `content/index.md` to modify this homepage.
3. Add new `.md` files in `content/` to expand your network.
4. Run `hwaro build` to compile the grid.
5. Run `hwaro serve` to preview changes locally on your personal terminal.
