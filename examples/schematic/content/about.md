# About Schematic

Schematic is a hardware and circuit design documentation site built with [Hwaro](https://github.com/hahwul/hwaro). It demonstrates structured technical documentation with spec tables, pin descriptions, and electrical characteristics -- the kind of content found in professional datasheets and hardware reference manuals.

## Purpose

This example showcases how Hwaro can serve as a platform for electronics engineering documentation, including component datasheets, reference designs, and integration guides. The clean, CAD-inspired layout prioritizes readability and precise data presentation.

## Features

- Weight-based page ordering within documentation sections
- Table of contents for long-form technical content
- Sidebar navigation for multi-page documentation
- Responsive design for lab and field use
- Search across all documentation pages
