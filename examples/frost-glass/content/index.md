# Frost Glass Portfolio

Welcome to the Frost Glass aesthetic. This design language focuses on dark, moody backgrounds with flowing, ethereal gradients, overlaid with translucent, frosted glass UI elements.

## The Aesthetic

The **glassmorphism** effect is achieved using backdrop filters, creating a sense of depth and hierarchy. Vibrant neon accents pierce through the blur, giving the interface a modern, cyber-elegance.

### Features
- **Dark Theme:** Deep, immersive backgrounds.
- **Glass Panels:** Frosted, translucent content containers using `backdrop-filter`.
- **Fluid Gradients:** Animated, glowing orbs and flowing colors that sit behind the glass.
- **Neon Accents:** Crisp, high-contrast borders and typography for readability.
