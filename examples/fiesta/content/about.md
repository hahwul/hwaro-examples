## About Fiesta

Fiesta is a vibrant theme inspired by Latin American celebrations — the papel picado banners, the marigold garlands, the pulsing rhythms that make every gathering unforgettable.

### Design Philosophy

This theme embraces **maximalism with purpose**. Every color, every gradient, every animation is chosen to evoke the warmth and energy of a real fiesta:

- **Papel Picado** — The iconic perforated paper banners that dance in the breeze above every celebration
- **Warm Palette** — Rich marigold, sunset coral, deep magenta, and tropical teal
- **Bold Typography** — Display headings that command attention, paired with clean body text
- **Dynamic Energy** — Subtle animations and gradients that bring the page to life

### Built With

Fiesta is built with [Hwaro](https://hwaro.hahwul.com), a blazing-fast static site generator written in Crystal. It requires no JavaScript frameworks and loads in a flash.
