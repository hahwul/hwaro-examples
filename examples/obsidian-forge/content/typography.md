title = "Typography as Structure"
description = "Type is not decoration. It is the load-bearing wall."
date = 2024-05-14
+++

A page is a building. The text is not paint applied to a finished facade. The text is the steel.

When typography is treated as ornament, the layout has to do work it was never designed to do. Boxes appear. Rules thicken. Background tints arrive to separate one block from the next. Each addition is a confession that the type is not carrying its weight.

A heading at thirty-two pixels in a strong cut already separates itself from a sixteen-pixel body. There is no need for a coloured bar to its left. There is no need for an icon. There is no need for an underline. The size and the weight are doing the work. Adding a bar is asking the reader to read the same signal twice.

Bodies, blockquotes, lists, captions, code. Each of these has a default voice. Inter at four hundred sets the body. Inter at six hundred sets emphasis. IBM Plex Mono at four hundred sets code and metadata. That is the entire vocabulary. A fourth voice will sound like a guest who was not invited.

Colour, in this system, is reserved. Pure white at the top of the contrast scale. A muted grey for secondary information. Nothing else. The eye learns the rule on the first paragraph and trusts it for the remainder of the page. Trust is the resource a long-form site spends down. Spend it carefully.

Lock the type stack early. Resist the temptation to introduce a third weight or a fourth size during the build. Each new value is another thing the reader has to learn before they can read.
