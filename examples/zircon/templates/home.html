Point your package manager at it

Zircon speaks the registry protocols your tools already know. Set a registry URL, drop in a token, and every install resolves through your mirror — provenance checked on the way in.

Configure a mirror → 
zircon registry set pkg.internal.example
zircon login --token $ZIRCON_TOKEN
zircon add @acme/http-kit
zircon verify @acme/http-kit --provenance
