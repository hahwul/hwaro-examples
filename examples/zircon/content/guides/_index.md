These four guides take a Zircon instance from a fresh binary to a registry
your whole organization depends on: install and point a package manager at
it, mirror the upstreams you already use, sign every publish with a
provenance attestation, and scope tokens so publish rights are never
shared secrets. Read them in order the first time through.
