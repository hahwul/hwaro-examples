## About Orchid

Welcome to Orchid — where exotic botanical beauty meets refined digital design. This site is built with [Hwaro](https://github.com/hahwul/hwaro), a fast and lightweight static site generator written in Crystal.

## About This Theme

The **Orchid** theme draws inspiration from the exotic elegance of orchid flowers. Rich purples, vibrant pinks, and soft magentas create a glamorous reading experience that balances bold visual impact with refined usability.

### Design Features

- Deep purple-to-magenta gradient backgrounds evoking twilight gardens
- Orchid pink and fuchsia glow effects on headings and interactive elements
- Floating petal animations on the homepage for botanical ambiance
- Shimmer hover effects on cards with soft gradient borders
- Elegant serif headings paired with crisp modern body text
- Responsive design optimized for all devices
- Syntax highlighting with Monokai theme

### Philosophy

Like the orchid itself — rare, striking, and endlessly varied — this theme embraces the idea that beauty and functionality are not opposites. Every gradient, glow, and animation is crafted to enhance the reading experience while delivering maximum visual impact.
