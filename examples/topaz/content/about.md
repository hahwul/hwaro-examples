## About Topaz

Topaz is a glamorous blog theme inspired by the golden warmth of topaz gemstones. With its rich amber tones, sunlit crystal facets, and dynamic visual effects, it delivers a bold and trendy reading experience.

### Design Philosophy

The theme embraces visual spectacle while maintaining readability:

- **Golden Warmth** — Rich amber and honey gradients evoke sunlit crystals
- **Dynamic Effects** — Subtle animations and glowing accents add life to every page
- **Bold Typography** — Elegant serif headings paired with clean body text
- **Crystal Facets** — Geometric patterns and light refractions in the design details

### Built With

This theme is built with [Hwaro](https://github.com/hahwul/hwaro), a fast static site generator written in Crystal. Pure CSS handles all visual effects — no JavaScript frameworks required.
