description = "Fundamental concepts of the framework."
+++

The framework is built on three main pillars:

1. **Reactivity**: High-performance state management.
2. **Modularity**: Every component is a self-contained unit.
3. **Extensibility**: Easily add new features via plugins.

For low-level utilities used by the framework, see the [Toolkit package](/toolkit).
