Solar Wind is the largest synthesis update since the 4.0 rewrite. The headline
is a wider FM engine: instruments now carry **six operators** instead of four,
which unlocks the classic stacked-carrier brass and bell timbres that four
operators could only approximate. Existing patches are safe — a four-operator
instrument loads into operators one through four with the other two disabled, so
old songs sound identical.

<!-- more -->

## Added

Six-operator patches are defined the same way as before, with two extra
operator blocks. A minimal bell:

```json
{
  "name": "glass-bell",
  "algorithm": 5,
  "feedback": 3,
  "operators": [
    { "ratio": 1.00, "level": 99, "attack": 31, "decay": 12, "sustain": 0 },
    { "ratio": 3.51, "level": 74, "attack": 31, "decay": 18, "sustain": 0 },
    { "ratio": 7.00, "level": 58, "attack": 28, "decay": 22, "sustain": 4 },
    { "ratio": 14.0, "level": 40, "attack": 25, "decay": 30, "sustain": 0 },
    { "ratio": 1.00, "level": 62, "attack": 31, "decay": 40, "sustain": 8 },
    { "ratio": 2.00, "level": 35, "attack": 31, "decay": 44, "sustain": 0 }
  ]
}
```

The new **ratchet** effect `Rxy` retriggers the current row `x` times, dropping
volume by `y` steps each hit — the fast machine-gun snares chiptune leans on,
without hand-placing note-offs.

## Changed

The pattern grid now numbers rows in hex to match the effect column. If you
prefer decimal, `View → Row Numbers → Decimal` flips it per project. The
envelope editor scrubs live: hold a note, drag a breakpoint, and you hear the
change on the sustained voice instead of on the next trigger.

## Fixed

The feedback operator clicked whenever its frequency ratio crossed an octave
boundary mid-note; the phase is now continuous across the jump. Sample loop
points imported from `.wav` files no longer reset to the full clip after a
save and reload.
