Dust Loop is the build the long-term-support channel tracks, so it favors
stability over new toys. The sample loader learned proper loop modes, the effect
column got readable, and a stack of small pattern-editor annoyances went away.

<!-- more -->

## Added

Samples now support three loop modes instead of one: **forward**, **ping-pong**,
and **one-shot**. Ping-pong is the one people kept asking for — it plays a short
grain out and back so a two-frame waveform holds a sustained pad without an
audible seam. Set the mode in the sample panel or per-note with `Sxx`:

```text
S00   one-shot   ; play once, ignore loop points
S01   forward    ; classic loop from start to end
S02   ping-pong  ; alternate direction each pass
```

## Changed

The effect column was rewritten to show a short mnemonic next to the hex code
while the row is focused, so `A08` reads as `A08 vol-slide` until you move off
it. It is the same two characters in the file — this is purely a display aid for
learning the command set.

## Fixed

Muting a channel during playback used to cut its release envelope abruptly;
mutes now fade over a couple of milliseconds. Renaming an instrument no longer
scrolls the list back to the top, and the row cursor stays put when you resize
the window.
