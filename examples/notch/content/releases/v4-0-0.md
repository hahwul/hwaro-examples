Hard Cut is a major version because it breaks two things on purpose: the audio
engine is new from the phase accumulator up, and the project format is new to
match. The payoff is clean high notes, sample-accurate automation, and a
command-line exporter you can wire into a build.

<!-- more -->

## Added

Notch now writes the `.nt` format — a single zipped bundle holding patterns,
instruments, and samples, with a plain-text manifest you can diff. Old `.mod`
and `.xm` files still import.

The exporter is scriptable. Render a song to WAV from a shell, no window:

```bash
notch render song.nt --out master.wav \
  --rate 48000 --bits 24 --tail 2.0 \
  --oversample 2
```

`--tail` holds the mix open so long release envelopes ring out instead of
getting chopped at the last row.

## Changed

Every FM operator is now **band-limited**. The old engine summed raw sine
partials, which aliased ugly harmonics into anything above the fifth octave;
the rewrite integrates each operator against the Nyquist limit, so a lead two
octaves up stays clean. Automation runs per-sample rather than per-row, which
means filter sweeps and volume ramps are smooth instead of stepped.

## Removed

The old fixed 8-channel limit is gone — songs can address up to 64 channels.
The legacy per-row-only automation path was removed; projects that used it are
migrated on first open, and a one-time report lists anything that changed.
