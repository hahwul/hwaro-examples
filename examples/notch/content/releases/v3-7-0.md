First Light is where the public history starts. Notch had been a private tool
for two years; this is the build we felt comfortable handing to other people. It
is deliberately small — four-operator FM synthesis, a keyboard-driven pattern
editor, and a WAV exporter — but everything in it works and has not changed shape
since.

<!-- more -->

## Added

The core is a **four-operator FM** voice with eight selectable algorithms and a
per-operator envelope. If you have used a hardware FM synth the model will be
familiar: operators modulate each other according to the chosen algorithm, and
feedback on the first operator adds grit.

The pattern editor is entirely keyboard-first. Notes go in from the home row,
effects from the hex keypad, and the whole grid navigates with the arrows:

```text
Notch 3.7.0  |  pattern 00  |  bpm 150  spd 06

ROW  CH01        CH02        CH03        CH04
00   C-4 01 .. .. E-4 02 .. .. .-. .. .. .. G-3 03 .. ..
01   .-. .. .. .. .-. .. .. .. .-. .. .. .. .-. .. .. ..
02   D-4 01 .. .. .-. .. A06   .-. .. .. .. .-. .. .. ..
```

## Known limits

This build caps songs at eight channels and exports only 16-bit WAV. Both
limits are lifted later in the 4.0 rewrite. There is no MIDI input yet and no
sample playback — Notch 3.7 is a pure synthesis instrument. The file format is
the plain `.mod`-compatible layout, which import keeps reading in every version
since.
