Glass Arp is the last release on the 3.x engine, and it is all about playing in.
You can drive Notch from a MIDI keyboard now, watch each channel on an inline
oscilloscope, and build fast arpeggios from reusable tables instead of the old
three-note `0xy` chord effect.

<!-- more -->

## Added

**Arpeggio tables** are named sequences of semitone offsets that a note walks
through, one step per tick. They loop until the note ends:

```text
arp MAJ7   = 0 4 7 11
arp OCTAVE = 0 12
arp STAB   = 0 3 7 | 0        ; the bar resets to root, then holds
```

Trigger a table with the `Jxx` effect. Because the table is separate from the
pattern, retuning a whole song's arps is a one-line edit.

MIDI input maps note-on to the focused channel, with velocity routed to the
volume column. Hold `Ctrl` while a controller moves to bind it to the parameter
under the cursor.

## Changed

Each channel header now carries a small **oscilloscope** that draws that voice's
output in real time. It is the fastest way to spot a patch that is quietly
clipping or a carrier that decayed to silence three rows ago.

## Fixed

Notes entered from the computer keyboard while playback was running sometimes
landed one row late; input is now quantized to the row the playhead is on when
the key goes down. The scope no longer freezes when a channel is muted and
unmuted quickly.
