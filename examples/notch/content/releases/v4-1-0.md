Neon Drift is a quality-of-life release aimed at people who write long songs.
The pattern editor no longer stutters when you scroll through a 4,000-row order
list, and the instrument panel finally shows you what each patch actually is
without opening it.

<!-- more -->

## Added

**Groove tables** let a pattern swing without touching timing by hand. A groove
is a short list of tick counts that the playhead cycles through, so a straight
sixteenth pattern can shuffle just by assigning it a groove:

```text
groove SHUFFLE = 4 2 4 2      ; long-short, long-short
groove PUSH    = 3 3 4 2      ; leans into the backbeat
```

Assign one with the `Txx` effect, where `xx` is the groove index. Setting `T00`
returns the channel to the song's global speed.

## Changed

The instrument list was redesigned around a compact two-line row: name on top,
a live algorithm glyph and operator count beneath. You can read a whole bank at
a glance instead of hovering each slot. Sorting is stable now, so dragging a
patch never reshuffles the ones around it.

## Fixed

Copying a selection that crossed a channel boundary used to drop the effect
column of the last channel. Paste now preserves every column. The metronome no
longer drifts out of phase after a tempo change mid-playback, and the waveform
preview redraws correctly when a sample is replaced in place.
