# About Glam Rock
Glam Rock is a bold dark theme inspired by 1970s glam rock with metallic accents and electric energy.
## Features
- Responsive layout for all devices
- Custom color palette and typography
- Clean post cards with hover effects
- Tag-based content organization
- RSS feed support
## Getting Started
```bash
hwaro init my-site --scaffold https://github.com/hahwul/hwaro-examples/tree/main/glam-rock
```
