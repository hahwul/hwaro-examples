# About Formulary

Formulary is a mathematical and scientific formula reference built with Hwaro. It serves as a demonstration of how a static site generator can present technical, formula-heavy content using KaTeX for LaTeX rendering.

## Purpose

This project demonstrates:

- Rendering LaTeX mathematics in a static site using KaTeX
- Organizing formula references with a sidebar-based documentation layout
- Presenting theorems, definitions, and proofs in structured formats
- Building searchable academic reference material

## Technology

Formulary uses [KaTeX](https://katex.org/) for fast, high-quality math typesetting. All formulas are written in standard LaTeX notation and rendered client-side. The site is generated by [Hwaro](https://github.com/hahwul/hwaro), a fast static site generator.
