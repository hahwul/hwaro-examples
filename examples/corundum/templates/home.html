What Corundum Is

Standards, not shortcuts

Corundum implements OAuth 2.1 and OpenID Connect as written. Implicit and password grants are gone; authorization code with PKCE is the only interactive path. Every deviation from the specification is documented and defaulted off.

Keys you can hold

WebAuthn is a first-class authenticator, not an afterthought. Roaming security keys and platform passkeys enrol through the same ceremony, and administrators can require a hardware factor per client.

Yours to run

A single static binary, a Postgres database, and a signing key. No control plane phones home; no tenant data leaves your host. The threat model in the Security Model assumes you, and only you, hold the root of trust.
