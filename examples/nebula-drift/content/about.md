# About Nebula Drift

Nebula Drift is a beautiful theme crafted specifically for the Hwaro static site generator.

## Features

- **Dark Mode First**: Carefully chosen dark color palette to reduce eye strain.
- **Glassmorphism Elements**: Translucent backgrounds with subtle blurs to create depth and hierarchy.
- **Neon Accents**: Bright, glowing highlight colors for interactive elements.
- **Responsive**: Fully responsive design that looks great on mobile, tablet, and desktop screens.
- **Lightweight**: Pure CSS, no heavy frameworks required.

This project showcases how flexible Hwaro templates can be.
