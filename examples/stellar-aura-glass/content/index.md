# Stellar Aura Glass

Welcome to a new dimension of web design. This theme utilizes **glassmorphism** over deep space gradients to create a celestial, futuristic aesthetic.

## Features

- **Glass Panels**: Frosted glass effects with backdrop filters.
- **Neon Accents**: Vibrant cyan and magenta gradients.
- **Deep Space Vibe**: Subtle radial gradients on a dark canvas.
