# About Cipher

Cipher is a cryptography reference documentation site built with Hwaro. It provides concise, accurate information about cryptographic primitives and protocols used in modern software systems.

## Purpose

This project serves as an example of a dark-themed technical documentation site. The design draws from terminal aesthetics and the visual language of security tooling.

## Credits

- **Hwaro** -- The static site generator powering this site
- **Inter** -- The sans-serif typeface used for body text
- **Fira Code** -- The monospace typeface used for code and headings
