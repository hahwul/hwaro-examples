# Welcome to Primer

Primer is a hands-on introduction to programming. Each lesson follows a clear structure: learn the concept, study a working example, then practice with an exercise.

Whether you are writing your first line of code or brushing up on fundamentals, these tutorials will guide you step by step using Python.

## Tutorials

- [Basics](docs/basics/) -- Variables, control flow, and functions
- [Patterns](docs/patterns/) -- Error handling and collections

## How to Use This Site

1. Start with the **Basics** section and work through each lesson in order.
2. Read the concept explanation carefully before looking at the code.
3. Try every exercise on your own machine before checking the solution.
4. Move on to **Patterns** once you are comfortable with the fundamentals.
