This template was created for the Hwaro static site generator.

It demonstrates how to use pure CSS to create complex, animated backgrounds and sophisticated UI elements like glassmorphism.

## Technologies Used

- **Hwaro**: The static site generator powering this site.
- **CSS Grid/Flexbox**: For layout and positioning.
- **CSS Animations**: For the floating orbs and moving grid.
- **Backdrop Filter**: For the frosted glass effect.
- **Space Grotesk & Inter**: Fonts from Google Fonts.

Here is an example of some code formatting:

```css
.glass-panel {
    background: rgba(28, 15, 50, 0.25);
    backdrop-filter: blur(16px);
    border: 1px solid rgba(157, 78, 221, 0.15);
    border-radius: 24px;
    padding: 3rem;
    box-shadow: 0 8px 32px 0 rgba(0, 0, 0, 0.6);
}
```
