# About the Theme

**Aurora Glass** is a demonstration of how flexible and expressive [Hwaro](https://github.com/hahwul/hwaro) templates can be.

By combining minimal HTML structure with modern CSS techniques like `backdrop-filter` and CSS animations, we can create immersive, app-like experiences for static sites.

## The Glassmorphism Trend

Glassmorphism is a UI trend characterized by:

- Translucency (frosted glass effect)
- Multi-layered approach with objects floating in space
- Vivid colors to highlight the blurred transparency
- A subtle, light border on the translucent objects

This theme implements these principles to provide a striking, modern backdrop for your content.
