# Experience the Aurora

Welcome to **Aurora Glass**, a modern, dark-themed static site template built with [Hwaro](https://github.com/hahwul/hwaro).

This template features a unique visual aesthetic combining deep, animated radial gradients with frosted glass elements (`backdrop-filter`) to create a floating, multi-layered experience.

## Key Features

- **Animated Background**: Soft, shifting aurora blobs created entirely with CSS.
- **Glassmorphism UI**: Translucent content containers that let the background colors shine through softly.
- **Modern Typography**: Utilizing the *Outfit* font family for a clean, contemporary look.
- **Responsive Design**: Adapts beautifully to mobile, tablet, and desktop screens.

## Getting Started

1. Edit `content/_index.md` to change this page.
2. Add new `.md` files in `content/` to create new pages.
3. Run `hwaro serve` to preview changes locally.
