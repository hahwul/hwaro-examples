# Measurement Reference

The Ammeter reference describes a small, repeatable procedure for recording current draw across the elemental circuits used on the bench. The point is consistency, not precision beyond what the meter can deliver.

## Probe Placement

Place the probe in series, not parallel. Verify the lead is intact and the contact point is clean. A loose contact will produce values that drift downward over the first few seconds of a measurement, which is easy to mistake for a real change in the circuit.

## Reading Window

Record three consecutive readings spaced ten seconds apart. Use the median as the canonical value for the test. Outliers usually indicate a transient elsewhere in the system, often a switching supply that has not fully stabilized.

## Notation

Every entry in the reference log carries the date, the test bench identifier, the ambient temperature, and the median value. Optional columns record the operator and the instrument serial. The notation is deliberately compact so that long sessions remain legible when the log is reviewed weeks later.
