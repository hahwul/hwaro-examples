# About Stained Glass
Stained Glass is a colorful theme inspired by cathedral stained glass windows with rich jewel tones.
## Features
- Responsive layout for all devices
- Custom color palette and typography
- Clean post cards with hover effects
- Tag-based content organization
- RSS feed support
## Getting Started
```bash
hwaro init my-site --scaffold https://github.com/hahwul/hwaro-examples/tree/main/stained-glass
```
