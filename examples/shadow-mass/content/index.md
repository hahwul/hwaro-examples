## The Mass Manifesto

We believe the web has become too soft. Too smooth. Too eager to please. Buttons that hover, gradients that breathe, illustrations that wink — all of it polite, all of it forgettable.

This publication is the opposite. We print in mass. We stack our paragraphs like cinderblocks. We choose one ink color and we stay with it. We allow exactly one accent — and only when it carries meaning.

Read us if you want type that does not apologize. Subscribe if you want a feed that does not bend. Close the tab if you wanted a softer corner.

— *The Editors*
