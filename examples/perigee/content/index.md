Perigee is named for the point in the Moon's orbit where it sits closest to Earth — the one night in the cycle when the tide pulls hardest and the moon looks larger than it has any right to.

We build one watch: a 38 millimeter hand-wound case, oxidized by hand and finished in a workshop of five, holding a moonphase disc regulated closely enough that it will not visibly drift from the real sky within a normal owner's lifetime. Every run totals three hundred pieces. Run One sold out over eleven months without an advertisement; Run Two started the following spring, cut on the same mill to the same tolerances, because a watch that changes between runs is a different watch, not an improved one. When a run sells out, the next one waits for a new bar of silver and an open bench — never for a faster factory.
