The numbers below describe Run Two exactly as it ships — the same sheet that travels with each movement from the bench where it's regulated to the case where it's fitted. If a figure changes between runs, it changes here first.

## Case

| | |
|---|---|
| Diameter | 38mm |
| Thickness | 10.4mm |
| Lug-to-lug | 44mm |
| Material | Fine silver, hand-oxidized |
| Crystal | Box-domed sapphire, anti-reflective inner coating |
| Case back | Sapphire display window, edge-engraved with run number |
| Water resistance | 30m (rain and washing, not swimming) |
| Crown | Push-pull, signed, 3 o'clock |

## Movement

| | |
|---|---|
| Caliber | P-38, in-house, hand-wound |
| Power reserve | 46 hours |
| Frequency | 21,600 vph (3Hz) |
| Jewels | 21 |
| Accuracy | −4/+6 seconds per day, unregulated |
| Winding | Manual only, no automatic rotor |

## Moonphase complication

| | |
|---|---|
| Drive | 135-tooth moon wheel, finger-cam advance |
| Drift | 1 day per ~122 years against the true synodic month |
| Setting | Corrector at 4 o'clock, recessed, adjustable without a case opener |
| Aperture | Arched window at 6 o'clock, hand-painted star field |

## Dial and hands

| | |
|---|---|
| Dial | Oxidized silver, lathe-turned |
| Indices | Applied silver batons |
| Hands | Gilt brass |
| Lume | None — Perigee is a daylight and lamplight watch |

## Strap and edition

| | |
|---|---|
| Strap | Hand-stitched calf leather, quick-release spring bars |
| Buckle | Oxidized silver, matched to case |
| Edition | 300 pieces per run, individually numbered |
| Current run | Run Two, No. 1 through 300 |

Every completed watch is timed on the bench across six positions for five days before it leaves the workshop; a movement that falls outside the −4/+6 second window gets pulled and re-regulated rather than shipped and disclosed as an outlier. For the design reasoning behind the case and dial, see [the watch](/the-watch/); for why each run stays capped at three hundred pieces, see [Why Three Hundred](/journal/why-three-hundred/) in the journal.
