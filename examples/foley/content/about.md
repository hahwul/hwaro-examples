Foley is a weekly-ish podcast where the people who made a game sit down and tell you what actually happened, in the order it happened. Not a highlight reel — a postmortem, the way film crews use the word: the reconstruction that happens after the thing is already finished, when you can finally hear which sounds were real and which ones you added later to cover for what didn't work on the day.

The show is hosted by Adaeze Brannigan, a former QA lead who spent six years finding other people's bugs before deciding it was more interesting to ask why the bugs shipped anyway. Every episode is one conversation, recorded in a converted garage in the same week the guest agrees to it, edited for length but never for candor. If a launch went badly, the episode says so by name.

## How an episode is built

Each conversation follows the same four beats, in the same order, every time:

1. **The pitch** — what the game was supposed to be before anyone wrote a line of code.
2. **The number** — one metric that mattered more than the rest: wishlists, refund rate, day-one concurrent players, whatever actually decided the studio's next move.
3. **What broke** — the bug, the deadline, or the decision that the team would take back if they could.
4. **What they'd keep** — the one choice that turned out right, even if it didn't feel right at the time.

## Episode numbering

Every episode card on this site carries a three-digit arcade score counter — 001, 002, and up — because that's genuinely how we track the season internally: a running tally taped above the mixing desk, incremented by hand after every recording. The number never resets and never skips, even on [the finale](/episodes/undertow-keep/), which shipped as a five-person roundtable instead of a solo interview.

The whole run so far lives in the [episode archive](/episodes/), oldest first, same order as the tally.

## Pitching a postmortem

If you shipped something and are willing to talk about the parts that didn't work, write to the show with the game's name, the platform it launched on, and one sentence about the number that mattered most. Season two booking opens once the current run wraps.
