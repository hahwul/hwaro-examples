A METAR is a weather report written like a telegram: fast to file, dense to read, and completely opaque the first fifty times you see one. Nimbus takes real observations off the wire and reads them back to you the way an instructor would in the run-up area &mdash; group by group, in plain English, with the "so what" for a light aircraft made explicit.

Every briefing below opens with the raw report it decodes. Skim the mono capsule, read the one-line translation beneath it, then dig into the full write-up if the sky looks interesting. New here? Start with [Reading Your First METAR](@/briefings/reading-your-first-metar.md), then browse [the full logbook](@/briefings/_index.md).
