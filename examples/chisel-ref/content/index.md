## Material Reference
Engraving the structural essence of refined information.

The chisel pattern strips a page back to its load-bearing pieces: a single rule, a single column, a single voice. Use it for technical references, glossary pages, and anything that benefits from being read in one sitting without sidebars or callouts pulling focus.

Each section sits inside a chisel-surface block bordered on the left by a black rule. The rule is the only ornament; the type does the rest. Set headings sparingly, prefer short paragraphs, and let the page breathe between them.

Pair this layout with a monospaced label face and a plain sans-serif for body copy. Avoid color. The contrast between the rule, the border, and the background is enough to mark hierarchy without adding visual noise.
