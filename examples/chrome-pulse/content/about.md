# About the Design

Chrome Pulse was created as a design exploration to capture a sense of "glamorous and trendy" without relying on common techniques like excessive gradients.

The design relies entirely on flat background colors and carefully calculated `rgba` box-shadows to generate a metallic, glowing "pulse" effect, creating a feeling of luxury and modernism.

## Technology

Built using the robust **Hwaro** static site generator. No external CSS frameworks were used, ensuring an incredibly fast, lightweight, and precise styling execution.
