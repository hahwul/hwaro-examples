Flagbearer is the workflow release. Everything in it exists so that closing your laptop on a six-pane session and reopening it feels like nothing happened.

<!-- more -->

## added

- **Session restore.** `semaphore --restore` (or `sessions.restore = "always"`) reopens every tab and pane with its layout, working directory, and the last 2,000 lines of scrollback. Running processes are not resurrected — that would be a lie — but the prompt reopens where the process lived.
- **Pane zoom.** `ctrl+shift+z` expands the focused pane to the full window; the same key restores the layout. The zoomed state survives tab switches and shows a `[zoom]` marker in the titlebar so you never lose panes behind it.
- **Prompt marks.** Shells that emit OSC 133 (fish does by default; zsh with a two-line hook) get prompt navigation: `ctrl+up` and `ctrl+down` jump the viewport between prompts, and `ctrl+shift+c` copies the full output of the command under the cursor.

```toml
[sessions]
restore = "ask"          # always | ask | never
scrollback_tail = 2000

[keys]
"ctrl+shift+z" = "pane:zoom"
"ctrl+up"      = "scroll:prev_prompt"
```

## changed

- New panes inherit the focused pane's working directory via OSC 7 instead of defaulting to `$HOME`.
- The tab strip elides pane titles from the middle, keeping the leaf directory visible.

## fixed

- Zoomed panes no longer swallow `ctrl+shift+arrow` resize events meant for the underlying layout.
