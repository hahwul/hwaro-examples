A small release that closes the three most-reported issues from [v2.1.0](/releases/v2-1-0/). If you saw ghost cursors on a discrete GPU, this is the one to install.

<!-- more -->

## fixed

- **Cursor trails on discrete GPUs.** The cursor quad was drawn from a staging buffer that could be one frame stale on drivers with three-deep swapchains. Discrete NVIDIA and AMD cards showed a faint trailing cursor during fast scrolls; the quad now sources from the current frame's uniform block. Reported independently eleven times — thank you for the captures.
- **Pane-resize flicker on X11.** Dragging a divider under X11 briefly presented a frame with the old pane geometry and the new viewport, which read as a white flash on light themes. Resize events now block presentation until the grid has re-wrapped.
- **Kitty keyboard protocol over slow links.** The progressive-enhancement handshake waited on a reply that some multiplexers swallow, stalling input for up to two seconds on high-latency SSH sessions. The handshake now times out at 50 ms and degrades gracefully.
- `semaphore doctor` no longer mislabels MoltenVK as a software renderer on macOS 15.

## notes

No config changes, no new features, no migration steps. v2.1.1 is a drop-in replacement for v2.1.0:

```console
$ semaphore --version
semaphore 2.1.1 (vulkan 1.3, shaper 2.1)
```
