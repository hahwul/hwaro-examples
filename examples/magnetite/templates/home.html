What the engine gets right

Typo tolerance

A Levenshtein automaton matches misspellings within a configurable edit distance, so altittude still finds altitude without a spell-check round trip.

Faceting

Declare a field as a facet and every response ships exact counts per value, computed from the same posting lists that answer the query.

Synonym maps

Two-way and one-way synonym sets expand at query time from a plain-text map, hot-reloaded without a reindex when the file changes on disk.
