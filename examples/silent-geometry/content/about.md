This project exists as a manifestation of minimalist web brutalism.

We utilize a strict grid layout, limited color palettes relying heavily on high contrast values, and robust typography to present information directly.

### The Stack

The foundation is built upon semantic HTML and pure CSS, served through Hwaro. The font utilized is **Inter**, selected for its exceptional legibility and austere geometric structure.

```css
/* Core ethos translated to code */
* {
    box-sizing: border-box;
    margin: 0;
    padding: 0;
}
```

By removing the extraneous, we are left only with the essential.
