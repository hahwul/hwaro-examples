Tuff is a small Python SDK with one job: get a hobby robot moving in the
first ten minutes, then stay out of your way. It speaks serial over USB to
desk-mounted arms and wifi to rolling rovers, using the same handful of
verbs for both. There is no IDE to install, no firmware toolchain to fight,
and no linear algebra prerequisite — `pip install tuff`, plug in, and the
first thing your robot does is wave at you.

The library ships with drivers for the boards hobby kits actually use, a
`tuff` command line for pairing and flashing, and a teach mode that lets you
program poses by moving the arm with your own hands. Everything in these
tutorials runs on a plain laptop — no cloud account, no telemetry phoning
home, no subscription. Just you, a robot, and a Saturday afternoon.
