# Neon Petals Theme

This directory contains the `neon-petals` Hwaro example site.

## Aesthetic & Design
- **Theme**: "Neon Botanical" - A Cyber-Botanical digital garden.
- **Palette**: Deep greens/blacks (`#020603`), neon pink (`#ff2a85`), neon green (`#00ff73`), and neon cyan (`#05d9e8`).
- **Typography**: Uses 'Space Grotesk' for headings and 'Outfit' for body text.
- **Effects**: Features ambient CSS radial gradients representing glowing petals/leaves, and utilizes `backdrop-filter: blur()` for glassmorphism panels.

## Structure
- `content/_index.md`: The root homepage section.
- `content/about.md`: A standard page.
- `static/style.css`: Contains all custom styling for the theme.
- `templates/`: Contains the base structure utilizing `header.html`, `footer.html`, and wrapped in `.glass-panel` components.
