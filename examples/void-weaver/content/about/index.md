## Forged in the Dark

The Void Weaver theme was created to showcase how beautiful dark UI and glassmorphism can be when combined with subtle animations. It is ideal for portfolios, personal sites, or creative showcases.

### The Toolkit

- **Typography:** Outfit for headings and Space Grotesk for body text.
- **Colors:** Deep abyssal black background contrasted with cyan, magenta, and purple glows.
- **UI:** Translucent panels casting subtle light to define depth.

### Getting Started

You can customize this theme easily by modifying the CSS variables in `static/style.css` and the template files in the `templates/` directory.

{% alert(title="Pro Tip") %}
Experiment with the `bgPulse` animation keyframes in the CSS to adjust the speed and scale of the background orbs to match your desired vibe.
