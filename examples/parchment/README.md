# Parchment Example

A fantasy RPG world-building guide and lore book theme for [hwaro](https://github.com/hahwul/hwaro).

## Features

- **Fantasy Aesthetic:** Styled like an old parchment or papyrus book.
- **Elegant Typography:** Uses "Cinzel" for headings and "EB Garamond" for body text.
- **Lore-Focused Structure:** Optimized for documenting races, regions, and history.
- **Light Theme:** A warm, readable light theme based on parchment tones.
- **No Gradients & Emojis:** A clean, professional fantasy look.
