# Parchment Agent Instructions

This site is a fantasy RPG world-building guide. When creating or modifying content, follow these rules:

## Content Style

- **Tone:** Use a formal, scholarly, or chronicler-like tone.
- **Terminology:** Employ fantasy-themed terms (e.g., "Lineages" for races, "Atlas" for regions, "Chronicles" for pages).
- **Structure:** Use clear headings, lists, and tables for technical details (like racial traits or location data).

## Design Constraints

- **No Gradients:** Do not use Tailwind gradient classes (e.g., `bg-gradient-to-r`). Use solid colors or simple borders.
- **No Emojis:** Do not use emojis in the content or UI. Use text-based separators or decorative border classes.
- **Typography:**
    - Headings: Use `font-display` (Cinzel).
    - Body: Use `font-serif` (EB Garamond).

## Templates

- `page.html`: Use for standalone pages like the home page.
- `section.html`: Use for lists of content (e.g., the races or regions landing page).
- `doc.html`: Use for detailed lore entries with a sidebar.
