# Welcome to Neo Vanguard

Experience a cutting-edge static site powered by [Hwaro](https://github.com/hahwul/hwaro).

This design merges sleek **glassmorphism** with vibrant **neon** accents over a deep void background, creating an elegant yet futuristic aesthetic.

## Features of this aesthetic

1. **Glassmorphic panels**: Translucent backgrounds with `backdrop-filter` for a frosted glass effect.
2. **Neon glows**: Cyan and magenta text-shadows and box-shadows.
3. **Deep void background**: A dark, immersive canvas with subtle, fixed radial gradients.
4. **Modern Typography**: The *Space Grotesk* font pairs perfectly with this cyberpunk vibe.

## Getting Started

- Explore the [About](/about/) page to see an individual page layout.
- Edit this content in `content/index.md`.
- Run `hwaro serve` to witness the future live.
