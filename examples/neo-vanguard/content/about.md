# About Neo Vanguard

This is the **Neo Vanguard** example for Hwaro. It demonstrates how to create a highly stylized, futuristic website using custom templates and simple markdown content.

## Philosophy

The goal of this design is to push the boundaries of what a simple static site can look like, utilizing modern CSS features like CSS variables, flexbox, grid, and `backdrop-filter`.
