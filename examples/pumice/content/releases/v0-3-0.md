The release that made Pumice useful with the tools teams already run: metrics
and traces now leave the agent as native OpenTelemetry.

<!-- more -->

### Added

- **OTLP/gRPC exporter.** Point `--endpoint` at any OpenTelemetry collector; no
  translation shim required.
- Low-cardinality trace export for sampled syscall and TCP events, with kernel
  stacks symbolized on the host.
- `syscall` and `tcp` collectors reach their first stable metric namespaces.

### Changed

- Probe ABI frozen as **v1**; later 0.x releases relocate against it without a
  reload. Kernels older than 5.4 are no longer supported.
- Config moved from flags-only to a `pumice.toml` file with flag overrides.

### Fixed

- Fixed a startup panic on hosts where `/sys/kernel/btf/vmlinux` was absent by
  falling back to a bundled BTF archive.
- Timestamps are now sourced from `CLOCK_MONOTONIC` to survive wall-clock steps.
