# About the Design

The **Chroma Origami** theme was born from a desire to combine contrasting aesthetics. Origami, traditionally an ancient art form relying on the simplicity of white paper and precise, unyielding folds, finds a new identity when splashed with vivid, modern color gradients.

## Inspiration

We drew inspiration from:
1. Classic *crane* and *lotus* origami patterns.
2. Synthwave and neon-noir color palettes.
3. Modern web design trends using CSS `clip-path` for non-rectangular layouts.

> "A folded corner is a promise of depth; a gradient is a promise of light."

## Technical Details

This site is powered by [Hwaro](https://github.com/hahwul/hwaro). The unique styling is achieved entirely through custom CSS variables and clipping paths, ensuring a lightweight and responsive experience across devices.
