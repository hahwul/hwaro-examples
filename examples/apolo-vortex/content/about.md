The **Apolo Vortex** design is built for Hwaro to demonstrate how modern CSS techniques can create a sense of depth and movement.

### Key Features

*   **Cosmic Background Animations:** Utilizing `conic-gradient` and `radial-gradient` combined with CSS animations.
*   **Glassmorphism:** Frosted glass panels achieved with `backdrop-filter: blur()`.
*   **Neon Accents:** Bright cyan (`#00f0ff`) and magenta (`#ff0055`) highlights against a deep void background (`#05030f`).
*   **Modern Typography:** Pairing `Outfit` for headings and `Space Grotesk` for body text.

This layout is perfect for portfolios, creative agencies, or any project that wants to leave a lasting, otherworldly impression.
