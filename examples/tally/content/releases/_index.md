Tally ships on a slow, deliberate cadence: features land only after the threat
model has been re-checked, and security patches are cut the day a fix is ready.
Each release records its exact version, its date, and whether it closed a
security issue. Subscribe to the [feed](rss.xml) to hear about patches first.
