# Enter the Vortex

Experience the deep space purples, glowing neon cyan and magenta accents, and frosted glass UI elements.

This is a demonstration of what is possible with Hwaro templates and a bit of custom CSS.

## Features

- **Glassmorphism:** Elegant frosted glass containers using `backdrop-filter`.
- **Fluid Gradients:** Animated, glowing backgrounds that feel alive.
- **Modern Typography:** Crisp Space Grotesk and Inter font stacks.
