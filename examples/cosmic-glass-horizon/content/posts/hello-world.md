Code blocks sit comfortably in deep, darkened containers with subtle glowing borders. Here's a quick example:

```python
def initiate_launch_sequence(planet):
    print(f"Preparing to launch toward {planet}...")
    for i in range(3, 0, -1):

    print("Liftoff! 🚀")

initiate_launch_sequence("Mars")
```
