# Manifestation

The Luminous Orb was created to demonstrate how far you can push Hwaro's templating engine and static site generation capabilities with a modern CSS approach.

By leveraging CSS variables, backdrop filters, and subtle animations, we transform a basic static page into an ethereal experience.
