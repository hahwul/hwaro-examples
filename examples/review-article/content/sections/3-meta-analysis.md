## Sensitivity Analyses

1. **Excluding high RoB studies (k = 18):** SMD = -0.78 [95 pct CI: -0.94, -0.62], confirming robustness
2. **Active control only (k = 8):** SMD = -0.54 [-0.74, -0.34], smaller but still significant effect
3. **Waitlist control only (k = 14):** SMD = -0.98 [-1.18, -0.78], larger effect as expected
4. **Therapist-supported dCBT-I (k = 10):** SMD = -0.94 [-1.14, -0.74]
5. **Fully automated dCBT-I (k = 12):** SMD = -0.72 [-0.92, -0.52]

The subgroup comparison between therapist-supported and fully automated dCBT-I showed a non-significant trend favoring therapist support (p = 0.08 for subgroup difference).
