NeoBrutal is a SaaS landing page theme built on neo-brutalism principles. Bold borders, hard shadows, clashing colors, and intentionally loud typography combine to create something that refuses to blend in.

This is not minimal. This is not subtle. This is a design language that demands attention and makes no apology for it.

Built with [Hwaro](https://github.com/hahwul/hwaro).
