Every drawing in this index starts as two things: a seed string and a function that fits on one screen. The function runs once against the seed, in whatever language that week's piece was written in, and it either draws the plot or it does not compile. Nothing gets nudged by hand afterward. The small sketch on each card below is drawn live, right now, in your browser — seeded with that project's own slug, using the same class of deterministic method described on its page. Reload and the lines will not move.

The physical drawings are made on a belt-driven XY plotter, technical fineliner pens on cold-press paper, one pass per ink. What gets published here is not a photograph of a finished piece — it is the seed, the function, the paper stock, and the pass count, so that anyone who wants the exact same drawing can run the exact same code and get it, this year or in ten.
