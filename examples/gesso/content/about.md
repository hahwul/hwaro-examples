Gesso is the plotter practice of Iris Halvorsen, a creative coder working out of a converted print shop where the presses used to be. The name is the primer coat you brush onto a panel before you paint it — the layer that decides how everything drawn afterward will sit on the surface. That is roughly what the code in this index does: it primes a seed, and the seed decides everything the pen will do.

The hardware is deliberately plain. A belt-driven XY plotter with two stepper motors and a servo lift, built from a kit and rebuilt twice since. Paper is almost always 300gsm cold-press stock cut to A4 or A3, occasionally a toothier printmaking sheet when a piece needs more drag on the nib. Ink is technical fineliner — 0.1mm for fine hatching, 0.3mm and 0.5mm for anything structural — plus the occasional gel pen pass when a series calls for a second color. No airbrush, no plotter that draws faster than the ink can dry.

The process is the same for every series published here. Write the function. Pick a name, slugify it, and use that slug as the seed — never a separate number, because the seed should be something a reader can type. Run the function once against the seed. If the plot is worth keeping, plot it once, on one sheet, and publish the function alongside it. If the plot is not worth keeping, change the function, not the seed, and try again. A piece is finished when the code and the drawing agree with each other, and it stays finished — the function attached to a published piece never changes again, so the drawing can always be regenerated exactly, on this machine or any other, by anyone who bothers to read the code.

There is no edition size printed on anything here, because the constraint was never scarcity. Anyone with a plotter, the right paper weight, and ten minutes can produce the identical drawing from the published seed. What is not reproducible is the specific sheet that came off this specific machine on the day it was plotted — the ink pooling half a millimeter wider at one corner, the paper's own grain catching the nib. That sheet is the only thing that is actually one of a kind, and it is not for sale; it is scanned, described, and kept in a flat file two rooms from this desk.
