# About Disco Fever
Disco Fever is a groovy dark theme inspired by 1970s disco culture with mirror balls and neon.
## Features
- Responsive layout for all devices
- Custom color palette and typography
- Clean post cards with hover effects
- Tag-based content organization
- RSS feed support
## Getting Started
```bash
hwaro init my-site --scaffold https://github.com/hahwul/hwaro-examples/tree/main/disco-fever
```
