# About Neon Abyss Mirage

The **Neon Abyss Mirage** is a Hwaro example site demonstrating the power of modern CSS and static site generation.

This theme was built to showcase how to create a highly atmospheric, dark-themed website with fluid animations and a striking visual hierarchy. It is perfect for portfolios, creative blogs, or any digital project that demands a futuristic, cyberpunk aesthetic.

### Project Details

- **Framework**: Hwaro
- **Style**: Custom CSS with Glassmorphism
- **Fonts**: Space Grotesk (Headings), Inter (Body)
- **Palette**: Deep Void (`#03010A`), Neon Cyan (`#00f0ff`), Magenta (`#ff0055`), Purple (`#8a2be2`)

Feel free to use this as a base for your own creations!
