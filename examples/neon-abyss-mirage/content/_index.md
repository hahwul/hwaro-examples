# Welcome to Neon Abyss Mirage

Step into the glowing void. This design utilizes a deep background inspired by the abyss and vibrant neon mirage effects. The combination of translucent glassmorphism panels and animated luminous orbs provides an elegant, cyberpunk, and modern portfolio layout.

## Features

* **Glassmorphism UI**: Beautifully blurred translucent containers that react dynamically to the background.
* **Animated Elements**: Floating neon orbs that create a mesmerizing, infinite void effect.
* **Cyberpunk Color Palette**: Vibrant cyan, magenta, and deep purple glows set against an abyss-black backdrop.
* **Space Grotesk & Inter**: Clean and highly legible typography that fits the futuristic vibe.
