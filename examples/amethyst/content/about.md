## About Amethyst

Welcome to Amethyst — a place where code meets crystalline elegance. This site is built with [Hwaro](https://github.com/hahwul/hwaro), a fast and lightweight static site generator written in Crystal.

## About This Theme

The **Amethyst** theme draws inspiration from the mystical beauty of amethyst gemstones. Deep violet hues, shimmering gradients, and crystalline effects create a glamorous reading experience that balances visual spectacle with readability.

### Design Features

- Deep purple-to-black gradient backgrounds with crystalline depth
- Violet and magenta glow effects on headings and interactive elements
- Shimmering card hover animations with faceted gem-like borders
- Gradient dividers that evoke light refracting through crystal
- Animated floating particles on the homepage
- Responsive design optimized for all devices
- Syntax highlighting with Monokai theme

### Philosophy

Like the amethyst stone itself — bold yet refined, vibrant yet calming — this theme proves that maximum visual impact and excellent readability can coexist. Every glow, gradient, and animation serves a purpose.
