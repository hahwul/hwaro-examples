The Silent Obelisk was forged as an experimental structure within the Hwaro ecosystem. It deliberately avoids the trappings of modern "soft" web design.

There are no gradients. There are no playful emojis. There are only solid lines, stark contrasts, and uncompromising text.

### Material Syntax

```html
<div class="monolith">
    <div class="void"></div>
    <div class="structure">Content</div>
</div>
```

The underlying ethos is that content should dictate its own container, and the container should not try to outshine the content.
