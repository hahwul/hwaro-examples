We design under the constraints of absolute structure.

This site is a curated architectural exercise. It applies the classical principles of the **International Typographic Style** (widely known as Swiss Design) to modern static web publishing. 

We strip away the decorative noise of the modern web—no visual gradients, no transient emoji sets, no database lookups. Instead, we present raw typography, clear hierarchical scaling, and asymmetric balance.

---

## core design constraints

1. **mathematical asymmetry**: We align content strictly to structural grid columns, utilizing unequal width divisions to create visual tension and flow.
2. **typographic dominance**: Fonts are set in high-contrast scales, allowing extra-bold sans-serif titles (Inter) to immediately organize visual priority.
3. **primary color focus**: Visual emphasis is created strictly using stark flat black panels and crimson red solid accents.
4. **static immutability**: Pages compile in milliseconds and are served as immutable, optimized assets.
