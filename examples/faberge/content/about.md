# About Faberge
Faberge is a luxurious theme inspired by the jeweled craftsmanship of the House of Faberge.
## Features
- Responsive layout for all devices
- Custom color palette and typography
- Clean post cards with hover effects
- Tag-based content organization
- RSS feed support
## Getting Started
```bash
hwaro init my-site --scaffold https://github.com/hahwul/hwaro-examples/tree/main/faberge
```
