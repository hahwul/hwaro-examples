This project is part of the **Hwaro** examples collection. It showcases the flexibility of the Hwaro static site generator to create wildly different visual experiences without being constrained by rigid theme structures.

### Architecture

The site uses a simple structure:
- `templates/base.html`: (In this simplified example, we use `header.html` and `footer.html` includes instead for straightforward composition).
- `templates/page.html`: Renders individual markdown files like this one.
- `templates/section.html`: Used for the homepage to list any potential sub-pages or posts.

### Typography

We use two variable fonts from Google Fonts to achieve this look:
1. **Space Grotesk**: Used for headings, navigation, and stylistic accents. Its geometric, slightly technical feel perfectly matches the cyberpunk aesthetic.
2. **Outfit**: Used for body copy. It is clean, legible, and pairs well with the bold headings.
