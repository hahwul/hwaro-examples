Welcome to the **Cosmic Synth Forge**. This is a standalone Hwaro example site demonstrating how to build a highly stylized, dark-themed, holographic UI using plain HTML and CSS, orchestrated by the Crinja templating engine.

### The Aesthetic

This design leverages several modern CSS techniques:

* **Glassmorphism**: Translucent panels using `backdrop-filter: blur(16px)` to create a frosted glass effect over the background.
* **Animated Orbs**: Background elements utilizing `radial-gradient` and keyframe animations to simulate floating cosmic energy.
* **Neon Accents**: Glowing text shadows and gradient text fills mimicking a futuristic synthwave environment.

### Getting Started

To use this template, simply run:

```bash
hwaro init my-site --scaffold github:hahwul/hwaro/examples/cosmic-synth-forge
```

Or just copy the `templates` and `static/css` directories into your own Hwaro project.
