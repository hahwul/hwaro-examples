This poster design was created for a local synthpop band's weekend gig. It heavily features the "bacterio" print and classic squiggles that define the Memphis look.

The primary colors used are hot pink (`#ff2a6d`) and cyan (`#05d9e8`), contrasting sharply with the thick dark navy borders.
