Welcome to the **Cyber Bloom Glass** theme for Hwaro. This design merges neo-cyberpunk aesthetics with glassmorphism, creating a visually striking digital experience.

## Features

- **Deep Void Background:** A dark canvas (#010206) sets the stage.
- **Animated Bloom Orbs:** Glowing cyan, magenta, and purple orbs drift smoothly in the background.
- **Frosted Glass Elements:** Translucent panels utilize CSS `backdrop-filter` for a sleek, modern look.
- **Elegant Typography:** Inter and Space Grotesk fonts deliver a clean, futuristic vibe.
