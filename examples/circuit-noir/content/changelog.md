The complete history of the OHMIC toolchain. We do not edit this file. Mistakes stay in the changelog.

## 2.04.07 — 2026-05-18

- **fix**: SPI transfer would stall on CS deassert under hard-disabled IRQs on Cortex-M0.
- **fix**: `ohmic trace --replay` skipped the last frame in the buffer.
- **docs**: Added the [Timing Diagrams](docs/timing/) reference with measured numbers from the lab.

## 2.04.06 — 2026-05-11

- **feat**: New `ohmic trace` subcommand for live UART monitoring with replay.
- **feat**: `ohmic doctor --board` now checks USB permissions on Linux without requiring root.
- **fix**: Inconsistent capitalization in CLI help output.

## 2.04.05 — 2026-04-29

- **fix**: GPIO pin 7 misclassified on STM32G0 footprint variants. The cargo feature `stm32g0-pin7-workaround` is no longer needed; remove it from your `Cargo.toml`.
- **fix**: `ohmic flash` would prompt for confirmation even with `--yes` on certain Windows shells.

## 2.04.04 — 2026-04-12 *(breaking)*

- **break**: Removed deprecated `legacy_uart()` helpers. Migrate to `uart_v2`. Migration guide in the [Protocol Layer](docs/protocol/) reference.
- **break**: `ohmic.toml` keys `chip_family` and `chip_variant` are merged into a single `chip` field.
- **feat**: `ohmic new` now produces a `memory.x` linker script tuned for the selected board.

## 2.04.03 — 2026-04-02

- **feat**: Added support for RP2350, including all PIO state-machine bindings.
- **feat**: New `--probe-list` flag on `ohmic flash` lists all connected debuggers.
- **fix**: ARM Cortex-M0+ trap handler would not save the FPU state register on parts without FPU.

## 2.04.02 — 2026-03-25

- **fix**: Timer overflow at 4hr29min on cortex-m0 baseline. Caused incorrect `ohmic::millis()` results on long-running firmware.
- **fix**: I²C SDA pin was not pulled high during initialization on STM32G071.

## 2.04.01 — 2026-03-12

- **fix**: First release after 2.04.0. Three small fixes; nothing user-visible.
- **fix**: `ohmic doctor` on Windows would fail with confusing message if WinUSB drivers were not installed.

## 2.04.0 — 2026-02-28

- **feat**: Initial release of the 2.04 series. Support for 142 MCU parts.
- **feat**: New protocol layer (`ohmic::proto::*`).
- **feat**: Async/await support throughout the HAL.
- **break**: Minimum Rust version is now 1.74.
- **break**: The `take()` peripheral pattern is now `take()?` (returns `Result`).

## 2.03.x — earlier

Earlier releases are archived. The full history is in the git repository.

## a note on deprecation

We deprecate features for a full minor version before removing them. A feature deprecated in 2.03.5 will be removed no earlier than 2.04.0. We do not remove features in patch releases.

We will never remove a feature without offering a migration path.
