A flag is easy to create and hard to release well. These six guides cover the
patterns Mica teams reach for most: how percentage bucketing stays consistent,
how to target a cohort before you touch the dial, when to trip a kill switch,
and how to read a rollout while it is happening. Each guide carries an on/off
pill — flipped **on** when the pattern is safe to ship as documented, **off**
when it needs a guardrail in place first.
