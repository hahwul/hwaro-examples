flint — a deterministic 2D physics module. Same inputs, same bytes, every machine, every replay.

flint runs on a single fixed timestep with integer-anchored math, so a simulation started from the same state produces identical output on any platform. That is what makes lockstep multiplayer and frame-perfect rollback possible.
