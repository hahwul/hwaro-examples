# About Nexus

Nexus is an API gateway and microservice orchestration platform built for modern distributed systems. This documentation site is built with [Hwaro](https://github.com/hahwul/hwaro) and demonstrates structured technical documentation with endpoint references, architecture diagrams, and service dependency visualization.

## Key Features

- **Service Mesh Integration** -- Native support for Envoy, Linkerd, and Istio sidecars
- **Intelligent Routing** -- Content-based routing, weighted traffic splitting, and canary deployments
- **Observability** -- Distributed tracing with OpenTelemetry, Prometheus metrics, and structured logging
- **Security** -- mTLS between services, JWT validation, OAuth 2.0 / OIDC integration
