Body text is for the argument. Display type is for the headline. Mono is for the metadata: dates, tags, counts, file paths, reference codes. It is a working typeface for working labels, and it keeps the rest of the page quiet.

Signal / White uses `JetBrains Mono` at 11px, with a little tracking, for anything a reader is meant to *check* rather than *read*. That is the rule. It has not needed amendments.
