A responsive rig should survive trends, which means it should have as few moving parts as possible. Four breakpoints is enough. Twelve is confusion in a trench coat.

## The four

- **Small** (under 600px): single column, stacked, generous leading.
- **Medium** (600-900px): eight-column feel, two-up cards.
- **Large** (900-1200px): twelve-column grid, comfortable.
- **Wide** (over 1200px): twelve-column grid, with air.

Beyond these, you are decorating. Decoration is not responsive. Decoration is a second job that did not need doing.
