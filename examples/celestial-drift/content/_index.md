# Celestial Drift

Embark on a journey across the stars. This theme is built on top of [Hwaro](https://github.com/hahwul/hwaro) and focuses on the intersection of modern aesthetics and fast performance.

> "To confine our attention to terrestrial matters would be to limit the human spirit."

## Features of this Theme

1. **Aurora Background**: A soothing, animated CSS radial-gradient background that shifts seamlessly.
2. **Glassmorphism Elements**: Frosty, translucent cards and headers achieved using `backdrop-filter`.
3. **Clean Typography**: High contrast and elegant sizing for pleasant reading in dark mode.

## Sample Code

Here's an example of how code looks against the dark theme:

```python
def find_stars(galaxy):
    stars = []
    for system in galaxy.systems:
        if system.is_luminous():
            stars.append(system.star)
    return stars
```
