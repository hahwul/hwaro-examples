Install day

Four visits' worth of work, done in one.

  1. 01

    Mounting rail

    A wall bracket rated for a full six-module stack goes up first, whether you are buying one module today or six.

  2. 02

    Base module

    The inverter, radio, and battery-management board land on the rail and run a self-test before anything else is wired.

  3. 03

    Modules

    Each module locks into place above the last with a quarter-turn latch. No tools, no exposed terminals once it is seated.

  4. 04

    Commissioning

    A full charge-discharge calibration, an app pairing, and a breaker walkthrough — the stack is live before the crew leaves.
