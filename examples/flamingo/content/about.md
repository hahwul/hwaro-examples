# About Flamingo

Flamingo is a glamorous and trendy blog theme for [Hwaro](https://github.com/hahwul/hwaro), a fast static site generator written in Crystal.

## Design Philosophy

This theme draws inspiration from:

- **Tropical resort aesthetics** — warm, inviting, and effortlessly stylish
- **Hot pink monochromatic palette** — bold, eye-catching, and unapologetically vibrant
- **Playful design accents** — gradients, rounded shapes, and dynamic visual layers

## Features

- Responsive layout that looks great on all devices
- Bold typography with the Poppins font family
- Eye-catching gradient hero section
- Clean post cards with hover effects
- Tag-based content organization
- RSS feed support

## Getting Started

To use this theme, install [Hwaro](https://github.com/hahwul/hwaro) and scaffold a new site:

```bash
hwaro init my-site --scaffold https://github.com/hahwul/hwaro-examples/tree/main/flamingo
```
