# About the Prism Structure

The **Obsidian Prism** design system was created to challenge the necessity of organic shapes in digital interfaces.

## Typography

We rely exclusively on two typefaces to maintain our structural integrity:
- **Inter:** Used for general reading and interface elements.
- **Space Mono:** Used for structural elements, technical definitions, and absolute values.

## Code Example

```html
<div class="prism-structure">
    <header class="prism-header">Absolute Form</header>
    <main class="prism-core">No Gradients</main>
</div>
```

The strictness of these parameters allows for a pure, unadulterated aesthetic experience.
