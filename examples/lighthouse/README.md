# Lighthouse: Navigation Guide / Directory

A curated resource directory and navigation guide theme for Hwaro.

## Design

- **Theme**: Light, clean blue/slate theme with clear information hierarchy.
- **Typography**: Inter / system sans-serif.
- **Layout**: Sticky header, hero section, searchable resource grid with filtering.
- **Features**: 
    - Card-based grid for resource browsing.
    - Real-time client-side search filtering.
    - Alphabetical index for quick navigation.
    - Rating (1-5 stars) and Recommended badges.
    - Category-based organization (`tools/`, `learning/`, `community/`).

## Styling

This example uses **Vanilla CSS** embedded in the `header.html` template. No external CSS frameworks are required, keeping it lightweight and easy to customize. Font Awesome is used for icons via CDN.

## Build

```bash
cd lighthouse
hwaro build
```
