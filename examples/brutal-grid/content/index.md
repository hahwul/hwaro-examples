Welcome to **Brutal Grid**. This site represents a radical departure from mainstream, over-decorated web designs. We embrace raw HTML structure, massive typography, and deep structural grids.

Here, you will find experiments in clean typography, grid theory, crystal programming, and static site systems. All layouts are designed with pure, flat colors and thick structural borders.

---

## core design principles

1. **heavy typography**: Font sizes should reflect structural hierarchy immediately. No subtle hints.
2. **thick solid borders**: Content boundaries are explicit, solid, and non-negotiable.
3. **zero gradients**: Light is simulated through pure flat colors and deliberate stark offsets.
4. **zero emojis**: Communication is achieved through pure unicode text elements, precise vocabulary, and clean visual layouts.
