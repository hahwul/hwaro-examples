# About

This site is built on the principle that the web does not need to be beautiful to be effective. Raw structure. Honest markup. No tricks.

Built with [Hwaro](https://github.com/hahwul/hwaro), a fast static site generator written in Crystal. The design draws from brutalist architecture: exposed materials, structural honesty, and a rejection of decorative excess.

## Principles

- Content is the interface
- Borders define space, not shadows
- Black and white. No compromise
- System serif fonts. No external dependencies
- If it does not communicate, it does not belong
