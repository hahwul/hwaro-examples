# About Scaffold

Scaffold is a project bootstrapping tool designed for developers who value consistent project structure and repeatable setup.

## Design Principles

- **Convention over configuration** -- Sensible defaults that work out of the box
- **Minimal dependencies** -- Ships as a single binary with no runtime requirements
- **Template portability** -- Template packs are plain directories that can be versioned and shared
- **Predictable output** -- The same template and variables always produce the same result
