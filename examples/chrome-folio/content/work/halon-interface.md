# Halon Interface

A control surface for a distributed fire-suppression system, designed for operators working in partial darkness. The display uses only three values — off, present, critical — so that status can be read at a glance from the far end of a corridor.

## Constraints

The interface runs on a six-inch panel mounted at knee height, visible only during emergencies. It must be legible from three meters away, decodable in under two seconds, and translatable to any of seventeen languages without layout changes.

## Response

We set the primary display in a single monospaced face at 64 pixels, reduced the palette to three values, and removed every optional affordance. The only animation is a one-frame flash on state change.

## Outcome

Operators in pilot installations reported a forty-two percent reduction in time-to-acknowledgment on simulated events. The panel shipped in eleven facilities in its first year.
