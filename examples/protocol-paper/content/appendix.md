## Ethics Statement

This study was approved by the Uppsala University Ethical Review Board (ref. 2025-0847, approved 22 September 2025). Reciprocal ethics agreements were established with the University of Toronto Research Ethics Board, the National Institute for Educational Research Ethics Committee (Tokyo), and the University of Cape Town Human Research Ethics Committee. All participants provide written informed consent prior to enrollment. The study is classified as minimal risk (educational intervention with no physical or psychological harm beyond normal coursework).

## Data Availability

The following materials are publicly available on OSF (osf.io/clarity-2026):

- Protocol document (all versions, PDF)
- Statistical analysis plan (R script with comments)
- Randomization code (R script, reproducible with seed)
- Sample size simulation script
- Item bank specification (without items, to preserve exam integrity)
- SPIRIT checklist (completed)
- Informed consent template
- Data collection instruments (SPQ-R, demographic questionnaire)

The final de-identified dataset will be deposited in the Uppsala University Data Repository within 12 months of the primary results publication. Access will be granted to qualified researchers upon reasonable request and completion of a data use agreement.

## Code Repository

Analysis code will be version-controlled on GitHub throughout the study. The repository will be made public at the time of data lock:

- Primary analysis scripts (R 4.3, lme4 package)
- Multiple imputation code (mice package)
- Tipping point analysis code
- Figure generation scripts
- Reproducibility manifest (renv lockfile)
