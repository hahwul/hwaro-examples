Six letters, six routes, one migration season. The Mailbag follows a different flyway or herd route in each issue — cranes on the Central Flyway, wildebeest circling the Serengeti, a godwit crossing open ocean without landing once — and files a short dispatch from wherever the data says the animals actually are.

There is no house style beyond honesty about distance. Every issue carries the coordinates it started and ended between, the window it moved in, and the species doing the moving, so a reader can lay six letters end to end and see a season of the planet's traffic at once.

The letters run in the order they were filed, oldest at the top of the path below, so the season reads the way it happened — a crane leaving Nebraska in January, a herd already circling the Serengeti by February, a bird crossing open ocean by May, and by June a caribou herd closing out the run on foot. New readers can start anywhere, but the path is built to be read from the top.
