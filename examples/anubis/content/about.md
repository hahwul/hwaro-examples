## About Me

I'm a writer and developer. This site is intentionally minimal — no JavaScript, no tracking, no distractions. Just words.

## About This Site

This is a demonstration of the **Anubis** theme for Hwaro, inspired by the [hugo-theme-anubis](https://github.com/Mitrichius/hugo-theme-anubis). It embraces brutalist minimalism: clean typography, generous whitespace, and nothing more.

### Principles

- Zero JavaScript
- No build pipeline
- ~200 lines of CSS
- Content first, always
