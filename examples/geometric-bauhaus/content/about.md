Geometric Bauhaus is an architectural and artistic exploration translating early-20th-century art principles into modern static build environments.

We believe that structure, form, and primary color can communicate complex information with absolute clarity.

## the visual rules

* **Color Space**: Bauhaus Cream `#FAF6EE` background, deep charcoal `#1E1E22` for text and borders, with flat primary red `#CC0000`, primary blue `#0F4C81`, and ochre yellow `#F3A322` accents.
* **Geometry**: Sharp rectangular borders, thick black dividing lines, and physical CSS geometric circle shapes.
* **Typography**: Geometric expressive titles (Syne) and clean modern body text (Outfit).

## compiler parameters

This project is built using **Hwaro**, a crystal-compiled static site generator. The template compilation cycles complete in under 10 milliseconds, outputting a zero-dependency, ultra-lightweight directory of pure static files.
