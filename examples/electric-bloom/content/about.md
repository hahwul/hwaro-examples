## About Electric Bloom

Welcome to Electric Bloom — where digital botanicals meet neon luminescence. This site is a celebration of bold, glamorous design that pushes the boundaries of visual expression.

## Design Philosophy

Electric Bloom draws inspiration from bioluminescent organisms and electric neon signage. Every element is crafted to create a sense of otherworldly beauty — glowing petals that pulse with energy, colors that bloom from darkness, and typography that radiates light.

### Key Design Elements

- **Neon gradients** that shift between magenta, violet, and electric cyan
- **Glowing text effects** inspired by bioluminescent flora
- **Dark canvas backgrounds** that let colors truly pop
- **Animated bloom transitions** on interactive elements
- **Bold typography** that commands attention
- **Rich visual layers** with depth and texture

## Built With

This site is powered by [Hwaro](https://github.com/hahwul/hwaro), a fast and lightweight static site generator written in Crystal. The entire visual design is achieved with pure CSS — no JavaScript frameworks required for the effects.
