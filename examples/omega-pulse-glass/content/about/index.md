This is the about page for the **Omega Pulse Glass** template. It demonstrates how a standard content page renders within the glassmorphism layout.

## Design Philosophy

The Omega Pulse theme was designed to create a sense of infinite depth. By utilizing a near-black void background combined with large, blurred, animated color orbs, the interface feels dynamic yet unobtrusive.

The content containers float above this void using the "glassmorphism" technique—semi-transparent backgrounds with a strong blur filter, creating a frosted glass effect that lets the pulsing colors shine through subtly.

### Tech Stack

- **Framework:** Hwaro Static Site Generator
- **Styling:** Pure CSS with custom variables and keyframe animations
- **Fonts:** Google Fonts (Space Grotesk & Inter)
