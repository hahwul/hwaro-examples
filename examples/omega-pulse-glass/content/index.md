Welcome to **Omega Pulse Glass**. A deep space void theme infused with pulsing neon orbs and premium frosted glassmorphism elements, crafted specifically for the Hwaro Static Site Generator.

## Features

- **Deep Void Background:** Rich dark `#03010A` environment.
- **Pulsing Neon Orbs:** Animated cyan, magenta, and purple radiant glows.
- **Glassmorphism Panels:** Translucent containers with `backdrop-filter: blur(16px)` and subtle frosted edges.
- **Modern Typography:** Combines *Space Grotesk* for headings and *Inter* for readable content.

{{ alert(type="info", body="This template is fully responsive and leverages modern CSS features like CSS Grid, Flexbox, and backdrop-filter.") }}

### Installation

```bash
hwaro init my-site --template omega-pulse-glass
cd my-site
hwaro serve
```

Experience the convergence of depth, light, and transparency.
