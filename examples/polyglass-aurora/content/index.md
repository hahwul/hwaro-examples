# Polyglass Aurora

Welcome to an elegant, high-performance static site built with [Hwaro](https://github.com/hahwul/hwaro).

## Ethereal Aesthetics

This template features a breathtaking Polyglass Aurora design, combining deep dark backgrounds with fluid, animated aurora borealis gradients and sleek frosted glassmorphism panels.

* **Aurora Background**: Seamless, flowing gradients powered purely by CSS animations.
* **Glassmorphism Panels**: Translucent containers with `backdrop-filter: blur(16px)` creating depth and elegance.
* **Modern Typography**: Utilizing the high-contrast *Outfit* for headers and *Inter* for highly readable body text.

## Performance First

Despite the rich visual effects, the underlying static site generated by Hwaro is blazing fast, providing an optimal experience without sacrificing aesthetics.
