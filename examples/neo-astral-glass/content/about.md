# The Architecture of the Void

The Neo Astral Glass template was forged in the depths of the static-site cosmos.

Its design philosophy relies entirely on the interplay between deep shadow and vibrant, diffused light. We use pure CSS features like `backdrop-filter` and `mix-blend-mode` to achieve depth without the overhead of heavy JavaScript libraries.

## The Stack

- **Hwaro:** The blazing fast static site generator.
- **Inter:** The geometric sans-serif for reading.
- **Space Grotesk:** The variable, bold sans for commanding headings.

> "The void is not empty; it is full of possibilities."
