# Enter the Void

Welcome to **Neo Astral Glass**, a deep void aesthetic featuring glowing animated orbs, translucent frosted glass panels (`backdrop-filter`), and sophisticated typography using Space Grotesk and Inter.

This is an example theme demonstrating how to build a highly stylized, modern landing page or portfolio using Hwaro.

## Features

- **Glassmorphism:** Elegant, frosted UI elements that float above the cosmic background.
- **Astral Animation:** Smooth, pure CSS radial gradients and blur filters creating an ambient, breathing background.
- **Minimalist Content:** Content shines brightly against the darkness, maintaining readability.
- **Typography:** Bold headings juxtaposed with highly legible body text.

[Explore the framework](https://hwaro.hahwul.com) to learn how to adapt this template for your own stellar nexus.
