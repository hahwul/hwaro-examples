# About Dojo

Dojo is a tutorial-style learning platform built with Hwaro. It demonstrates how to create educational content with structured learning paths, step-by-step navigation, and a clean reading experience.

## What You Will Learn

This platform covers everything from installing Hwaro to advanced configuration. Each lesson builds on the previous one, guiding you through a complete learning path.

## How to Use

Navigate through the lessons in order using the sidebar, or jump to any topic that interests you. Each lesson includes:

- **Difficulty level** -- so you know what to expect
- **Estimated duration** -- to help plan your study time
- **Code examples** -- with clear explanations
- **Navigation** -- to move between lessons seamlessly
