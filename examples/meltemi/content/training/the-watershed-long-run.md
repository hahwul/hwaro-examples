A slight pullback from peak week — forty-nine instead of fifty-five —
but Saturday's twenty-miler carried more weight than any run before it,
because Runa built it as a full dress rehearsal: race shoes I've now worn
exactly twice, the gels I've settled on after months of testing worse
ones, and the same 6am start time Myrtali's course director has planned
for September. Nothing new on race day is the whole idea, and this was
the last long run with enough distance left in the build to still change
anything about the plan.

<!-- more -->

It went well, with one exception: the second gel at mile eleven sat
wrong, a problem I hadn't had with it in shorter efforts, which means
either the heat or the cumulative fatigue of the block changes how my
stomach handles fueling on tired legs. Runa's response when I told her:
"Better to learn that in June than at kilometre eighteen in Myrtali."
Correct, if a little bracing to hear. I'm switching that gel out before
the taper starts.

Total: 49 miles. The whiteboard photo this week has Myrtali's date
written in the corner for the first time, underlined once: thirteen
weeks out, the last of the truly big mileage behind me, and the taper —
still mostly theoretical from here — waiting on the other side of July.
