This is **Monolithic Slate**, an elegant and brutalist template for Hwaro. It uses a stark, dark aesthetic with clean typography and defined borders.

## Design Philosophy
The design relies on simple structural elements, raw shapes, and high contrast.

- Deep slate background (`#0d1117`)
- Off-white typography (`#c9d1d9`)
- Accent color for interactive elements (`#58a6ff`)
- Monospace fonts for a structural, technical feel

This is an example paragraph to show how text renders. `Inline code` is also styled to match the theme.
