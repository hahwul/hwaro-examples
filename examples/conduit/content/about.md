# About Conduit

Conduit is a documentation example for Hwaro, designed to showcase a data pipeline documentation site with a teal-and-white color palette and a clean engineering workbench aesthetic.

## Design Principles

- **Clarity** -- Every page should communicate its purpose immediately
- **Consistency** -- Uniform structure across all pipeline stage documentation
- **Practicality** -- Real-world configuration examples that engineers can adapt
- **Completeness** -- Schema definitions, data types, and configuration references

## Credits

- **Hwaro** -- The static site generator powering this site
- **Inter** -- The typeface used throughout
- **Conduit** -- A fictional data pipeline tool created for demonstration purposes
