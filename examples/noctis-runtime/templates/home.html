// PRINCIPLES

Noctis is a streaming runtime, not a general-purpose batch system. Every operator is built around the assumption that events arrive continuously and out of order. The runtime does not retry, does not buffer beyond a bounded window, and does not silently coerce types.

If a behaviour is not in this reference, the runtime does not have it. The reference is the source of truth.
