This is the piece the board was renamed for. Candela ran under a different
name for its first year — the sysop's initials and a number, in the
tradition of the time — until GLASSWING uploaded this file to the ANSI art
door and someone in the message base suggested the board deserved a name
with light in it. Candela, a unit of luminous intensity, stuck.

The drawing itself is unremarkable in construction: forward and back
slashes for the wings, a parenthesis body, two lowercase Os for eyes. What
makes it work is restraint — GLASSWING left the wing tips open rather than
closing every line, so the shape reads as translucent even in a character
set with no shading glyphs at all. We found three later re-uploads of this
exact file under different filenames, each with a single line moved by one
column, which suggests it was retyped from a printout rather than copied
byte for byte. The version here is the earliest surviving copy, checksum
verified against the node's file listing from December 1994. GLASSWING
posted forty-one more pieces over the board's run and never used color
codes in any of them, even after the door software added support.
