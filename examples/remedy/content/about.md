# About Remedy

Remedy is a troubleshooting and debugging guide built with [Hwaro](https://github.com/hahwul/hwaro). It demonstrates structured documentation for technical support content, including symptom-cause-solution workflows, searchable error code indexes, and log analysis examples.

## How to Use This Guide

Each troubleshooting article follows a consistent structure:

- **Symptom** -- What you observe when the problem occurs
- **Cause** -- The underlying reason for the behavior
- **Solution** -- Step-by-step instructions to resolve the issue

Error code reference pages provide a comprehensive table of codes with severity levels, descriptions, and recommended actions.

## Features

- Weight-based page ordering for logical reading flow
- Table of contents for long reference articles
- Syntax-highlighted code blocks for log examples and commands
- Search across all troubleshooting content
