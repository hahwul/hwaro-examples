API keys are available on Pro and Enterprise plans:

1. Go to **Settings** then **API**
2. Click **Generate New Key**
3. Give your key a descriptive name (e.g., "CI/CD Pipeline")
4. Set the permissions scope (read-only, read-write, or admin)
5. Copy and store the key securely -- it won't be shown again

Each workspace can have up to 10 API keys. You can revoke any key at any time from the same settings page.
