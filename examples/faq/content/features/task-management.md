Tasks are the building blocks of every project. Each task can include:

- A title and detailed description with rich text formatting
- Assignees, due dates, and priority levels
- Labels and custom fields for organization
- File attachments and comments for collaboration
- Subtasks for breaking down complex work

Tasks can be viewed in multiple layouts: board (Kanban), list, timeline, and calendar. Drag and drop tasks between columns to update their status.
