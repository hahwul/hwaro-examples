Files can be shared across your workspace in several ways:

- **Task attachments:** Drag and drop files directly onto any task
- **Project files:** Upload to the project's shared file area
- **Comments:** Attach files to comments in tasks or discussions

Supported file types include documents, images, videos, and archives up to 100 MB per file (500 MB on Enterprise plans). All files are stored securely with encryption at rest and in transit.
