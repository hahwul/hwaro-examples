Automation rules help you eliminate repetitive tasks:

1. Go to your project settings and click **Automations**
2. Click **New Rule**
3. Define a trigger (e.g., "When a task is moved to Done")
4. Define an action (e.g., "Assign to reviewer" or "Send Slack notification")
5. Save and enable the rule

You can chain multiple actions on a single trigger and add conditions to control when the rule fires. Common automations include auto-assigning tasks, sending notifications on status changes, and updating due dates.
