Welcome to Emerald! This is your first post. Edit or delete it, then start blogging.

Emerald is a minimal blog theme built for Hwaro, based on the original [Emerald Jekyll theme](https://github.com/KingFelix/emerald) by Jacopo Rabolini.

## Features

- Clean, minimal design
- Mobile-first responsive layout
- Easy to customize
- Built-in syntax highlighting

Get started by editing the files in the `content/posts/` directory.
