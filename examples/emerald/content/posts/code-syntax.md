Emerald supports syntax highlighting for code blocks. Simply wrap your code in fenced code blocks with the language specified.

Here is a CSS example:

```css
body {
  background-color: #fff;
}

h1 {
  color: #ffaa33;
  font-size: 1.5em;
}
```

And this is a HTML example:

```html
<html>
  <a href="example.com">Example</a>
</html>
```

A JavaScript example:

```javascript
function hello() {
  console.log("Hello World!");
}

hello();
```

And a Python example:

```python
def hello():
    print("Hello World!")

if __name__ == "__main__":
    hello()
```
