date = 2025-04-12
description = "Embracing the raw texture of the web."
+++

The web does not need to be hidden behind layers of soft abstraction. In our design system, we treat HTML and CSS as raw materials—akin to poured concrete and exposed steel.

By revealing the underlying structure, we create an interface that is honest, durable, and uncompromising. Every border, every stark background choice, is an intentional decision to prioritize function and structural integrity over superficial ornamentation.
