Torque keyboards come in three footprints, and every one of them starts as
the same 1.4-kilogram bar of 6063-T5 aluminum. What changes between models
is the milling program, not the material or the process — a TQ-60 and a
TQ-90 are cut from the same stock, checked against the same tolerances, and
finished on the same line.

## TQ-60 — Sixty

No function row and no arrow cluster on the primary layer — everything
above the number row lives one key-hold away, mapped to the same positions
on every model so muscle memory carries over the moment you switch boards.
It's the one people buy for a second desk, or a first trip through an
airport, because it disappears into a laptop sleeve with room to spare.

| Spec | Value |
| --- | --- |
| Layout | 60%, 61 keys |
| Weight | 640g |
| Footprint | 296 × 104mm |
| Stack height | 18.4mm front / 22.0mm rear |
| Connection | USB-C, detachable cable |

## TQ-75 — Seventy-Five

Our best seller, and the board we recommend to anyone ordering their first
Torque. A full function row sits above the alpha block, and a dedicated
arrow cluster is pulled in tight against the right edge — no gap, no
navigation column eating desk space. It's the shape most people already
know how to type on.

| Spec | Value |
| --- | --- |
| Layout | 75%, 84 keys |
| Weight | 780g |
| Footprint | 330 × 132mm |
| Stack height | 18.4mm front / 22.0mm rear |
| Connection | USB-C, detachable cable |

## TQ-90 — Ninety

The TQ-90 adds a stacked navigation column — Home, End, Page Up, Page
Down — above the arrow cluster, and drops the numpad entirely. It's built
for people whose hands leave the alpha block more often for navigation than
for numbers: writers, reviewers, anyone who spends a day moving through
long documents instead of spreadsheets.

| Spec | Value |
| --- | --- |
| Layout | Tenkeyless with nav column, 96 keys |
| Weight | 860g |
| Footprint | 368 × 132mm |
| Stack height | 18.4mm front / 22.0mm rear |
| Connection | USB-C, detachable cable |

## Finish options

Every case is bead-blasted first, which is what gives the aluminum its
matte, non-reflective surface, and then goes into the anodizing tank in one
of three colors. Charcoal is the standing default and the fastest to ship;
Bone and Rust are batch-dyed and run on a rolling cycle tied to tank
capacity, not to demand — we anodize in whatever order fills the tank
efficiently, not strictly first-come-first-served.

| Finish | Description | Typical lead time |
| --- | --- | --- |
| Charcoal | Standard matte black-grey anodize | 1–2 weeks |
| Bone | Warm off-white anodize | 4–6 weeks |
| Rust | Oxide-red anodize | 4–6 weeks |
| Raw | Bead blast, no anodize | 1 week |

## Ordering and lead times

Boards are built in the sequence they're placed, in batches sized to
whatever's next on the mill's schedule. A Charcoal TQ-75 in stock finish
usually ships within two weeks. Anodized specials and larger batch orders
can run six to eight. We would rather quote the real number than a hopeful
one — see [Support](/support/) for how we handle delays past that window,
and what happens if a board arrives with a machining flaw we missed. Every
model runs on the same [Signal switch](/switches/) line, so a finish
decision is the only real choice left once you've picked a footprint.
