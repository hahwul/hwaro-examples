This is Intarsia, a high-contrast theme built with the Hwaro static site generator.

The Intarsia template provides:

- **Stark Contrast:** Pure whites and deep blacks to make text pop.
- **Bold Typography:** Uppercase headers and generous tracking.
- **Geometric Blocks:** Content encapsulated in thick, bordered blocks with offset shadows.

No gradients, no emojis—just pure, elegant form.
