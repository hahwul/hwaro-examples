# Column Count

Twelve columns is a lot of columns. Most editorial grids need four, perhaps six. Twelve is what you use when you want a spreadsheet that looks like a page.

Settle the count early. Draw the grid on the actual viewport before you pour in any content. If you can see where the text will sit without squinting, you have chosen too few columns and too large a body face. If you cannot find the text at all, you have chosen too many.

Six is usually correct. Six is rarely exciting. That is a feature.
