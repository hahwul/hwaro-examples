Obsidian Grid is a reading environment first. Typography carries the page; the grid is visible; the palette is a single axis from black to white. Every spread is cut by thin rules so the eye can settle, and the only ornament is a mark the writer actually drew.

The site is built with [Hwaro](https://hwaro.hahwul.com), a fast static site generator written in Crystal. Browse the [dispatches](/posts/) for long-form pieces, or scroll through [notes](/notes/) for shorter observations.
