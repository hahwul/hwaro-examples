# Colophon

This journal is set in **Space Mono**, a monospaced display face designed by Colophon Foundry for Google. Body copy is set solid, without italicization, to preserve the rhythm of the grid. Rule weights are fixed at one hairline.

## Grid

The page is built on a twelve-column grid with six fixed vertical rules drawn at the body background. The rules are deliberately visible. Captions and metadata flush to the leftmost rule; body copy extends across the inner nine columns; never more.

## Palette

Only two values carry meaning on this site: **#0a0a0a** for the field, and **#f4f4f4** for ink. A single muted gray is allowed for metadata. Accent colors are refused. If a passage needs emphasis, it earns it through rank — a heading, a rule, a blockquote — not chroma.

## Workflow

Entries are drafted in plain text, checked against a small grammar, then committed to a public repository. The site rebuilds on every merge. There is no comment section, no telemetry, no newsletter. If you have something to say, write a letter.
