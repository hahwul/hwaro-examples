We are exploring the boundaries of static site generation with a completely reimagined visual language. **Astral Zenith** introduces deep void backgrounds and animated glowing orbs directly within your Hwaro templates.

## The Aesthetic

The **Glassmorphism** effect is achieved using translucent panels:
```css
.panel {
    background: rgba(10, 5, 20, 0.4);
    backdrop-filter: blur(16px);
    -webkit-backdrop-filter: blur(16px);
    border: 1px solid rgba(255, 255, 255, 0.1);
}
```

## Features
* **Pure CSS Orbs**: Animated gradient circles that drift smoothly across the background.
* **Frosted Glass Components**: Seamless content readability without sacrificing the atmospheric depth.
* **Modern Typography**: Combining Space Grotesk for bold, structural headers and Inter for crisp, clear body text.
