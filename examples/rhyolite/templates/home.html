What the renderer handles for you

Retained scene graph

Build the tree once and mutate it across frames. A dirty flag propagates per subtree, so a node nobody touched since last frame is never re-tessellated or re-uploaded.

GPU tessellation

Cubic beziers flatten to an adaptive tolerance and cache by path identity and scale bucket, so panning and zooming re-tessellate a handful of times, not every frame.

Batched instancing

Shapes sharing a pipeline state draw in one instanced call. Ten thousand identical cells cost one draw call and one instance-buffer write, not ten thousand calls.

Compute-shader passes

Coverage accumulation and clip masks run as compute passes against small tile buffers, not stencil-buffer tricks — and the frame graph is open to your own passes.
