# Ethos: Engineering Principles & Culture

A documentation template for team culture and engineering principles. Focused on a light, professional aesthetic with a warm gray and coral color palette.

## Design

- **Theme**: Light theme with Warm Gray (#f5f5f4) and Coral (#ff6b6b) accents.
- **Typography**: Inter for sans-serif, Fraunces for serif headings.
- **Strict Rules**: **No gradients** and **No emojis**.
- **Features**: Principle cards with SVG icons, citation blocks, and decision-making flowcharts.

## Tailwind CSS

This example uses **Tailwind CSS Play CDN** (`cdn.tailwindcss.com`) for styling. Custom configurations for colors and typography are defined in the template.

## Build

```bash
cd ethos
hwaro build
```
