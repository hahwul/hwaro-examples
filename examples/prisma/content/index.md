Prisma is a running catalogue of what happens when ordinary light meets a cut edge. Every plate below began the same way: a beam crossed a boundary it wasn't built for — glass instead of air, a facet instead of a flat pane — and came out sorted into its component wavelengths. Runa Solheim has kept the bench since the winter of 2024, logging each specimen by angle, material, and the spread of colour it produced, then redrawing the result as a vector plate so the record can be re-measured by anyone who opens the file. Read the method in [the colophon](@/colophon.md), or start below with the newest specimen on file.
