# About Bulwark

Bulwark is a documentation example for Hwaro, designed to showcase a disaster recovery planning site with a dark theme using red, gray, and white color palette. The design evokes a command center atmosphere appropriate for incident response documentation.

## Design Principles

- **Readability** -- High-contrast text on dark backgrounds for extended reading sessions during incidents
- **Urgency** -- Red accents communicate the critical nature of disaster recovery operations
- **Structure** -- Hierarchical organization mirrors escalation paths and recovery procedures
- **Practicality** -- Every page contains actionable procedures, not just theory

## Credits

- **Hwaro** -- The static site generator powering this site
- **Inter** -- The typeface used for body text throughout
- **Noto Sans Mono** -- The monospace typeface used for code blocks and commands
- **Bulwark** -- A fictional disaster recovery framework created for demonstration purposes
