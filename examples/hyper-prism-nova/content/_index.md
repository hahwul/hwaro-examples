Welcome to **Hyper Prism Nova**.

This is a demonstration of a highly customized, futuristic glassmorphism aesthetic built for the Hwaro static site generator. It utilizes CSS variables, translucent panels, and animated neon gradients.

### Features
* Deep space ambient backgrounds
* Neon glowing accents (Cyan, Magenta, Purple)
* Refractive glassmorphism panels
* Custom futuristic typography
