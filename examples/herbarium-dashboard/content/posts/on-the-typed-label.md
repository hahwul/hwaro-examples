A herbarium is only as good as its labels. The pressed plant is the evidence; the label is the testimony. Lose one and the other is merely decoration.

{% alert(type="Field Note") %}
A determination written in pencil on the mounting card, with no locality and no date, is worth nothing to the next curator. Fill every field, every time.
{% end %}

## The fields of a proper label

Each card carries the same hand, in the same order, so the eye learns where to look:

1. **Family** &mdash; the higher grouping, set in small capitals.
2. **Genus and species** &mdash; the binomial, in italic, with the authority.
3. **Locality** &mdash; named station, then habitat in a phrase.
4. **Collected** &mdash; date and collector's initials.
5. **No.** &mdash; the accession number, never reused.

## Why the binomial is italic

The convention is not vanity. Setting *Digitalis purpurea* apart from the running text tells the reader, at a glance, that this is a name governed by the Code and not a common phrase. The family name stays upright; only genus and species lean.

> Print the number first and the name will follow it forever. Print the name first and the number will wander.

The catalog now runs past twelve hundred sheets, and not one of them leaves the cabinet without its card.
