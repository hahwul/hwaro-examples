# About Rave
Rave is a high-energy dark theme inspired by 1990s rave culture with acid colors and strobe effects.
## Features
- Responsive layout for all devices
- Custom color palette and typography
- Clean post cards with hover effects
- Tag-based content organization
- RSS feed support
## Getting Started
```bash
hwaro init my-site --scaffold https://github.com/hahwul/hwaro-examples/tree/main/rave
```
