## About This Site

This is a demonstration of the **Poison** theme for Hwaro, inspired by the [Hugo Poison](https://github.com/lukeorth/poison) theme by Luke Orth. It features a clean dark design with sidebar navigation and table of contents.

### Design Philosophy

- **Readability first**: Light text on dark backgrounds, generous spacing, and clean typography
- **Sidebar navigation**: Persistent navigation that stays visible while scrolling content
- **Minimal dependencies**: Pure CSS layout with no JavaScript frameworks
- **Two-column layout**: Sidebar for navigation, main area for content, optional TOC on wide screens

### Built With

This site is built with [Hwaro](https://github.com/hahwul/hwaro), a fast and lightweight static site generator written in Crystal.
