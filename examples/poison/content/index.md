Welcome to **Poison**, a clean dark theme with sidebar navigation for [Hwaro](https://github.com/hahwul/hwaro). Inspired by the [Hugo Poison](https://github.com/lukeorth/poison) theme.

## Features

- Fixed sidebar with hierarchical navigation
- Dark color scheme for comfortable reading
- Table of contents on wide screens
- Dot-leader post listings
- Responsive design with mobile hamburger menu
- Pure CSS, no JavaScript framework dependencies
- Syntax highlighting with Monokai theme
- RSS feed and tag support
