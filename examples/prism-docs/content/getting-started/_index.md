## Why Hwaro?

Hwaro is designed for speed and simplicity. It uses a straightforward directory structure and TOML-based configuration, making it easy for developers to focus on writing great documentation.

```bash
# Install hwaro (assuming you have the binary)
mv hwaro /usr/local/bin/
```
