This is a demonstration of the **Velvet Nebula** theme, crafted specifically for the Hwaro static site generator.

The goal was to create an atmosphere that feels expansive and serene, utilizing modern CSS properties like `backdrop-filter` and advanced keyframe animations. It is ideal for portfolios, personal blogs, or any digital space that requires a touch of cosmic elegance.

```css
/* Core colors */
--bg-color: #080315;
--accent-1: #c026d3;
--accent-2: #3b82f6;
```
