# About This Theme

Book is a documentation theme for Hwaro, designed for knowledge bases, wikis, and technical books.

It draws inspiration from the [Hugo Book](https://github.com/alex-shpak/hugo-book) theme and aims to provide a clean, distraction-free reading experience.
