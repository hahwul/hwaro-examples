## About

The Daily Gazette is an original theme for [Hwaro](https://github.com/hahwul/hwaro), inspired by the editorial layouts of classic broadsheet newspapers. Every element — from the masthead to the column rules — pays homage to the print tradition while embracing the possibilities of the web.

### Design Principles

- **Serif-first typography** with Playfair Display headlines and Lora body text
- **Multi-column layout** using CSS `column-count` for an authentic newspaper feel
- **Drop caps** on article openings via `::first-letter`
- **Small-caps and letter-spacing** for dates, bylines, and navigation
- **Cream paper background** (`#faf8f2`) with dark red (`#8b0000`) accents
- **Double-rule borders** echoing traditional broadsheet design

### Setup

1. Clone or download the files into your Hwaro project
2. Run `hwaro serve` from the terminal
3. Open your browser to `http://localhost:3000/`
