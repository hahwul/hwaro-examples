Set theory is the branch of mathematical logic that studies sets, which informally are collections of objects. Although any type of object can be collected into a set, set theory is applied most often to objects that are relevant to mathematics.

## Basic Definitions

A **set** is a well-defined collection of distinct objects. The objects that make up a set are called **elements** or **members** of the set. We write `a in A` to denote that `a` is an element of the set `A`.

### Notation

Sets are typically denoted by uppercase letters and their elements by lowercase letters:

- `A = {1, 2, 3, 4, 5}` -- a set of integers
- `B = {x | x is an even integer}` -- set-builder notation

## Key Operations

**Union**: The union of two sets `A` and `B`, written `A U B`, is the set of all elements that belong to `A`, to `B`, or to both.

**Intersection**: The intersection of `A` and `B`, written `A n B`, is the set of all elements that belong to both `A` and `B`.

**Complement**: The complement of `A`, written `A'`, is the set of all elements in the universal set that are not in `A`.
