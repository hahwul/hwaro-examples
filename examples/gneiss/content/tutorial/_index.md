Start here if you have never run a continuous query before. Each chapter builds on the last: the first gets a single query running against a live stream, the second explains how Gneiss treats graph updates as events rather than table rows, and the third covers windows, joins, and what happens when data shows up late. Read them in order, or jump straight to whichever problem you have right now.
