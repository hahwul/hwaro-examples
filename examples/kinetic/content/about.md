## About Kinetic

Kinetic is a motion-driven interactive agency site built for those who believe the web should feel alive. Every element on this site responds to physics -- gravity, inertia, elasticity -- creating an experience that feels tangible and dynamic.

## Design Philosophy

This site draws its atmosphere from the interior of a particle accelerator: clean lines, dark surfaces, and the quiet hum of energy in motion.

- **Velocity Red accents** -- drawing the eye like a laser beam through darkness
- **Momentum Black foundations** -- deep, rich backgrounds that let content breathe
- **Energy White typography** -- crisp, high-contrast text for effortless reading
- **Force Blue highlights** -- precision accents for interactive elements
- **Physics-based motion** -- every animation follows real-world dynamics
- **No decorative noise** -- clean geometry and purposeful spacing

## Technology

Built with [Hwaro](https://github.com/hahwul/hwaro), a fast and lightweight static site generator. The Kinetic theme uses the Web Animation API for physics simulations and CSS custom properties for a cohesive dark design system.
