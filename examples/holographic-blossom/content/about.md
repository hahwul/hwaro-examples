# About the Blossom

The **Holographic Blossom** was forged to blend the serenity of nature with the infinite potential of cyberpunk digital art.

Here, traditional layout rules are diffused through layers of frosted glass and vibrant neon light. The result is an interface that feels both futuristic and deeply organic.

### Features
- Deep void background (`#0a040b`)
- Animated, glowing holographic backdrops
- Custom CSS variables for easy theming
- Native dark-mode glassmorphism
