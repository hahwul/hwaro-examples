Commonplace opens its ledger with the smallest promise a note app can make:
your notes are files, on your disk, forever readable even if we vanish
tomorrow. No account, no server round-trip, no sync you didn't ask for. This
first folio lays the foundation everything else in this chronicle builds on.

<!-- more -->

### Added

- A vault is a folder of plain `.md` files with YAML front matter for title,
  created date, and tags. Nothing proprietary, nothing binary.
- `[[wikilink]]` syntax resolves to any note by title, case-insensitively,
  and creates the target note on first use if it doesn't exist yet.
- A single-pane editor with live markdown rendering — headings, lists,
  blockquotes, and code fences render as you type without a preview toggle.
- `commonplace open <path>` from the command line opens a vault directly,
  skipping the launcher entirely.

### Changed

- Nothing yet — this is the first folio.

### Fixed

- Nothing yet — this is the first folio.

A note on scope: 0.1.0 has no graph, no backlink panel, and no sync. Those
are coming, in that order, across the next three folios. We'd rather ship a
narrow tool that keeps its promise than a wide one that doesn't.
