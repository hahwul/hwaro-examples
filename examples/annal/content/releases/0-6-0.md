The sixth folio closes out this chronicle's first eighteen months with the
feature most requested in the backlog: search that actually understands what
you typed, plus a way to get the whole ledger out of Commonplace and into
anything else.

<!-- more -->

### Added

- Full-text search rebuilt on a proper inverted index instead of the old
  linear scan, supporting `tag:`, `title:`, and `-exclude` operators
  alongside fuzzy matching on the free text.
- `cp` — a small companion CLI, installed alongside the app, for scripting
  vault operations: `cp search "tag:graph -archived"`, `cp export --format
  bundle`, `cp watch` for a live change feed piped to stdout as JSON lines.
- Ledger export: `cp export --format bundle` writes the whole vault as a
  portable Markdown bundle with a `manifest.json` recording every backlink,
  so the graph survives the trip to another tool even without wikilink
  syntax support on the other end.
- A folio index in the sidebar — a compact chronological list of every note
  by creation date, styled after this changelog's own ledger rules, for
  vaults where browsing by time beats browsing by graph.

### Changed

- Search results now rank by a blend of text relevance and backlink count,
  so a heavily-referenced note edges out a barely-linked note on a tied
  text score — in practice this is usually the note you wanted.
- Graph View's incremental index (0.3.0) now backs search too, removing the
  second index that used to drift out of sync with it on very large vaults.

### Fixed

- Vaults over 50,000 notes took several seconds to open; startup indexing
  is now streamed and the UI is interactive before the last few thousand
  notes finish indexing in the background.
- `cp export` previously dropped tags on notes with no body text; empty
  notes now export their front matter intact.
- Daily notes (0.5.0) created past midnight in a timezone west of UTC could
  file under the wrong date; the date now resolves from the OS locale
  instead of UTC.

Query syntax reference for the new search:

```text
tag:graph               notes tagged "graph"
title:ledger            notes whose title contains "ledger"
-tag:archived           exclude archived notes
"exact phrase"          quoted phrases match literally
```

Six folios in, the ledger stays the plan: small, dated entries, nothing
rewritten after the fact. What shipped is what's written here.
