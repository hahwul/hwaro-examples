Classical brutalism in architecture used concrete because concrete was honest. The slab held the building up and showed you it was holding the building up. Web brutalism inherits that argument and then adds something the original movement refused: color, used as a structural member rather than as decoration.

## The Wall and the Paint

A flat lime panel next to a hot magenta panel is not a mood board. It is a load path. The eye uses the seam between them to locate the edge of a section, the same way a reader uses a horizontal rule. Remove the colors and the layout has to work harder, usually by adding more lines, more borders, more chrome.

```css
.section--alpha { background: #c8ff00; color: #000; }
.section--beta  { background: #ff2bd6; color: #000; }
.section--gamma { background: #00b3ff; color: #000; }
```

Three flat fills replace what a softer design would solve with twelve dividers and six shadows.

## Saturation as Hierarchy

Vibrant brutalism does not whisper. It also does not shout uniformly. The brightest hue belongs to the element that needs to be acted on: the primary button, the open call, the link out. Everything else steps down a notch in saturation, never in size. A page reads as a stack of equal-weight blocks where one is louder, and the user knows where to look without ever being told.

## Refusing the Tasteful

Tasteful neutrals are a way of admitting the layout is not strong enough to carry color. If the structure is honest, the color can be honest too. Cyan against orange, ruled with a five-pixel black border, on a yellow page. The grid is doing the work. The color is showing you the grid is doing the work.
