## The Concept

Neumorphism (or soft UI) focuses on elements that appear to push through the background, creating a physical, tactile interface using subtle light and dark shadows.

### Features

1. **Animated Background:** A slow, pulsing aurora gradient.
2. **Responsive Layout:** Adapts beautifully to mobile devices.
3. **Seamless Hwaro Integration:** Built on top of standard Hwaro templates.

This example was generated dynamically to demonstrate Hwaro's templating flexibility.
