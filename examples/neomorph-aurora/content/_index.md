Welcome to the **Neomorph Aurora** theme! This design blends soft neomorphic shadows with an ethereal, animated aurora background.

## Design Elements

The theme features custom styling for standard Markdown elements:

*   **Neomorphic Cards:** The main content area and listing items are styled as soft, extruded cards.
*   **Gradient Typography:** Headings utilize a vibrant, linear gradient text fill.
*   **Inset Code Blocks:** Code snippets appear pressed into the background.

### Code Example

```html
<div class="neomorph-card">
  <p>Soft UI meets Aurora gradients.</p>
</div>
```
