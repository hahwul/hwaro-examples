# Welcome to Luminous Echo

Explore a realm where deep voids meet echoing pulses of neon light. This theme, **Luminous Echo Glass**, brings forth an ethereal connection between dark backgrounds and vivid foreground elements.

## Design Highlights

- **Deep Void Background**: A gradient background diving from `#03010A` to `#080314`.
- **Luminous Orbs**: Smoothly pulsing cyan, magenta, and purple orbs that dynamically breathe life into the scene.
- **Glassmorphism Elements**: Translucent containers with `backdrop-filter: blur(16px)` and subtle glowing borders.
- **Modern Typography**: Bold `Space Grotesk` headings paired with highly readable `Inter` body text.

### Code Sample

```css
.glass-panel {
  background: rgba(255, 255, 255, 0.03);
  backdrop-filter: blur(16px);
  border: 1px solid rgba(255, 255, 255, 0.08);
  border-radius: 16px;
  box-shadow: 0 8px 32px 0 rgba(0, 0, 0, 0.4);
}
```
