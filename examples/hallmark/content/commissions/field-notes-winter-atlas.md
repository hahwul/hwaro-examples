Field Notes runs a themed cover series each year rather than a single design refreshed monthly, and this year's brief was Winter Atlas — four issues tracing a season from first frost to thaw. The problem was less "design a cover" than "design a system four different contributors can put a cover through and still land on."

<!-- more -->

I built the system around a single specimen block — the issue number, month, and a one-line field note set in a fixed position and scale on every cover, regardless of what sat behind it. That constant let the imagery underneath do whatever the issue needed: issue one ran a photograph from a contributing photographer's own frost archive, issue two ran a woodcut-style illustration commissioned specifically for the brief, and issues three and four ran type-only treatments when the imagery under consideration didn't clear in time. Because the specimen block never moved, none of the four covers reads as the odd one out on a newsstand shelf.

The type-only issues turned out to be the ones readers asked about most, which wasn't the outcome I'd predicted going in — I'd assumed the photography would carry the series. Field Notes has asked to keep the specimen system for next year's run with a different seasonal theme, which means the real deliverable wasn't four covers so much as a piece of scaffolding the magazine can keep using after I've moved on to the next brief.
