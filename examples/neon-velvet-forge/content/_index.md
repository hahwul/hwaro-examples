Welcome to the **Neon Velvet Forge**, a dark, immersive theme for Hwaro.

This theme features deep velvet backgrounds, glowing neon elements, and frosted glass panels to create a striking, modern aesthetic. It uses `backdrop-filter` for the glassmorphism effect and ambient animated radial gradients to simulate a living environment.

### Features
- **Deep Velvet Glassmorphism:** Translucent panels over a deep, dark void.
- **Neon Accents:** Bright cyan, magenta, and deep purple highlights.
- **Responsive Typography:** Using Space Grotesk and Inter for a clean, futuristic look.
