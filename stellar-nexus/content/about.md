## The Nexus Architect

Stellar Nexus is a high-performance static site built to test the limits of aesthetic design while maintaining optimal load times.

It was constructed using **Hwaro**, a lightning-fast static site generator written in Crystal.

### Technology Stack
* **Generator:** Hwaro
* **Design:** Custom CSS, Glassmorphism UI
* **Typography:** Space Grotesk (Headings), Outfit (Body)

We transmit data across the cosmos, curating elegant digital experiences.
