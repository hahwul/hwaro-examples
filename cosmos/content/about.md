## About Cosmos

Cosmos is a science magazine theme inspired by humanity's deepest venture into space. Each page is a window into the void -- dark, vast, and filled with points of light waiting to be discovered.

## Design Philosophy

- **Deep space palette** -- cosmic black, stardust silver, planet blue, ring gold
- **Starfield ambiance** -- subtle twinkling background stars
- **Magazine layout** -- clean typography for long-form scientific content
- **Dark, contemplative atmosphere** -- Voyager probe deep space exploration

## About the Theme

Built with [Hwaro](https://github.com/hahwul/hwaro), Cosmos is a dark magazine theme for science writing, astronomy, and space exploration content.
