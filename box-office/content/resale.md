## Joining the waitlist

Sign in with the same account you used on the original on-sale. Select the number of tickets you need (maximum four) and the tier preference. The waitlist timestamps your position to the second, so earlier sign-ups clear first when returns become available.

## How returns happen

Tickets can be returned by the original buyer up to 48 hours before each show. Returns occur for several reasons:

- Duplicate purchases flagged by the anti-scalping filter
- ID mismatches caught during pre-show verification
- Genuine personal circumstances (illness, travel, bereavement)
- Voluntary returns when a plan changes

The returned ticket re-enters the system at face value plus the standard processing fee. You pay only that amount &mdash; never more.

## Third-party sellers

We do not recognise tickets sold on secondary marketplaces, whether advertised above or below face value. These tickets cannot be validated at the door and the buyer is refused entry. The official resale waitlist is the only authorised route.

Report suspected scalping to fraud@meridiantheatre.example with a screenshot and the listing URL. Confirmed scalpers are banned from future on-sales across the entire venue group.
