Step into a vibrant digital landscape. Cyber Aurora Bliss combines the sleek aesthetics of retro-futuristic synthwave with modern, frosted glass UI components. Feel the gentle glow of the animated background as you explore the space.

## Features

- **Aurora Background**: Smooth, shifting CSS background gradients create an ethereal atmosphere.
- **Glassmorphism**: Translucent panels blur the content underneath for depth and layering.
- **Neon Accents**: Bold cyan, magenta, and purple colors reminiscent of classic cyberpunk and outrun styles.
- **Fluid Typography**: Carefully selected fonts (Outfit, Space Grotesk, and Inter) for striking headers and readable content.

## Quick Start

You can create new content simply by adding markdown files to the `content` directory:

```md
+++
title = "New Post"
description = "This is a new page showcasing the theme."
+++

Your glowing text goes here!
```

> **Note**: This theme is built with pure CSS and Hwaro's powerful templating engine.
