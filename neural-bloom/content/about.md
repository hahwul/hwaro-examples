# About Neural Bloom
Neural Bloom is a dark theme inspired by neural network visualizations and synaptic bloom patterns.
## Features
- Responsive layout for all devices
- Custom color palette and typography
- Clean post cards with hover effects
- Tag-based content organization
- RSS feed support
## Getting Started
```bash
hwaro init my-site --scaffold https://github.com/hahwul/hwaro-examples/tree/main/neural-bloom
```
