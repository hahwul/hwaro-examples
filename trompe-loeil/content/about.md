This template showcases a daring approach to web design, relying entirely on **flat colors, sharp contrast, and heavy geometric shadows** to create an illusion of depth—a modern *trompe-l'œil*.

There are strictly **no gradients** and **no emojis** used here. Just pure, brutalist-inspired CSS shapes that trick the eye into seeing three dimensions on a 2D screen.

### Features
* High contrast solid borders
* Monotone heavy drop shadows
* Folded corner CSS illusions
* Brutal, utilitarian typography
