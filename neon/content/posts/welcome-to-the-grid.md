Welcome to the grid. This is the first post on a site powered by [Hwaro](https://github.com/hahwul/hwaro) and the Neon theme.

## Why Neon?

The cyberpunk aesthetic isn't just about looks — it's about pushing boundaries. Neon colors against dark backgrounds create high contrast that's easy on the eyes during late-night coding sessions.

This theme uses pure CSS for all visual effects:

- `text-shadow` for the glow on headings
- `box-shadow` for card hover states
- CSS `linear-gradient` for the neon dividers
- CSS custom properties for easy color customization
