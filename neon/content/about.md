## About Me

Welcome to the neon grid. I'm a developer who lives at the intersection of code and creativity. This site is built with [Hwaro](https://github.com/hahwul/hwaro), a fast and lightweight static site generator written in Crystal.

## About This Site

This is a demonstration of the **Neon** theme for Hwaro — a cyberpunk-inspired design featuring neon glow effects, dark surfaces, and futuristic typography. Pure CSS, no JavaScript required for the visual effects.

### Features

- Deep purple background with neon pink and cyan accents
- Glow effects on headings, links, and interactive elements
- Card-style post listings with hover animations
- Gradient neon dividers
- Terminal-style code blocks with neon borders
- Responsive design for all devices
- Syntax highlighting with Monokai theme
