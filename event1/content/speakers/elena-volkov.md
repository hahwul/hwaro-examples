Elena Volkov is a Staff Frontend Engineer at DesignCraft and an invited expert on the W3C CSS Working Group.

She has been instrumental in the design and implementation of CSS Container Queries, one of the most requested CSS features of the past decade. Elena regularly contributes to browser engines and CSS specifications.

She is the author of "Modern CSS Architecture" and speaks at conferences worldwide about pushing the boundaries of what CSS can do.
