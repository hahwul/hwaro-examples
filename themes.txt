1. Neumorphism: Soft UI 및 그림자 효과 강조
2. Memphis: 80년대 멤피스 스타일 (볼드 컬러, 패턴)
3. FrutigerAero: 2000년대 중반 광택, 자연 친화적 UI
4. Y2K: 2000년대 초반 사이버/메탈릭 에스테틱
5. Bauhaus: 스위스 디자인, 원색 기하학적 레이아웃
6. Skeuomorphic: 현실적인 질감 (나무, 가죽 등)
7. ArtDeco: 1920년대 개츠비 스타일, 골드 라인
8. Victorian: 고딕, 화려한 고전 장식 및 세리프 폰트
9. Newspaper: 클래식 브로드시트 다단 신문 레이아웃
10. Typewriter: 타자기 스타일의 모노스페이스 미니멀리즘
11. RetroMac: 클래식 Mac OS 9 시스템 UI 디자인
12. Windows95: 클래식 Windows 95 회색 베젤 디자인
13. VintageTV: CRT 스캔라인, 레트로 브로드캐스트
14. CinemaSilent: 1920년대 무성 영화 자막 스타일
15. Vinyl: 레트로 레코드판 슬리브 디자인
16. Cassette: 아날로그 카세트 테이프 에스테틱
17. Steampunk: 황동, 톱니바퀴, 빅토리안 공상과학
18. Renaissance: 클래식 유화 액자와 우아한 세리프
19. Noir: 필름 누아르 스타일의 흑백 고대비
20. MiamiDeco: 파스텔톤, 네온, 마이애미 아르데코
21. Parchment: 고대 두루마리와 캘리그라피
22. Neoclassical: 대리석 질감, 조각상 레이아웃
23. Isometric: 아이소메트릭 3D 프로젝션 UI
24. ParallaxHeavy: 극단적인 스크롤 패럴랙스 애니메이션
25. Liquid: 액체 형태, Blob, 끈적이는 SVG 효과
26. Holographic: 홀로그램, 무지개 반사 포일 효과
27. ASCII: 모든 요소를 ASCII 아트로 렌더링
28. MinimalZen: 극단적 미니멀리즘, 여백의 미
29. Asymmetric: 비대칭 그리드, 겹치는 엘리먼트
30. PopArt: 강렬한 원색과 하프톤 도트
31. Claymorphism: 3D 점토 느낌의 UI 엘리먼트
32. CyberNinja: 하이테크 + 전통 동양 모티브
33. BioTech: 유기적 형태와 회로기판의 결합
34. SpaceStation: NASA 스타일 대시보드 및 원격측정 UI
35. Velvet: 고급스러운 진홍색 벨벳 텍스처
36. Mosaic: 스테인드글라스 또는 모자이크 타일 레이아웃
37. PixelArt: 8/16비트 레트로 게임 에스테틱
38. CyberGoth: 다크 사이버네틱, 인더스트리얼 고스
39. RetroFuturism: 70년대 크롬과 레트로 곡선의 미래상
40. Psychedelic: 60년대 사이키델릭 포스터, 일그러진 텍스트
41. Utopia: 밝고 자연 친화적인 솔라펑크 도시 느낌
42. Dystopia: 오염된 사이버펑크, 경고 사인
43. Whiteboard: 흰색 보드에 보드마카 손글씨 스타일
44. Glow: 모든 것이 은은하게 빛나는 다크 모드
45. StainedGlass: 대성당 창문 에스테틱, 형형색색의 투명 창
46. RetroSciFi: 1950년대 펄프 공상과학 잡지 커버
47. Surrealism: 초현실주의, 왜곡된 비율과 떠다니는 오브젝트
48. BrutalWeb: 로우 HTML 느낌의 의도된 안티디자인
49. Gilded: 금박, 정교한 세공 장식, 럭셔리
50. NeonWire: 빛나는 와이어프레임 벡터 그래픽
