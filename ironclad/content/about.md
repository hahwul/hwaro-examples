## About Ironclad

Ironclad is a security-focused product landing theme built with the weight and authority of reinforced steel. Every design decision reflects the uncompromising nature of defensive infrastructure.

## Design Philosophy

- **Gunmetal and rivet aesthetic** -- industrial materials inform every visual element
- **Military precision** -- structured layouts with deliberate hierarchy
- **Dark, authoritative atmosphere** -- warship combat information center feel
- **Zero ornamentation** -- every element serves a purpose

## About the Theme

Built with [Hwaro](https://github.com/hahwul/hwaro), Ironclad is designed for security products, defense technology, and industrial applications that demand visual authority.
