# Book

A clean, GitBook-style documentation theme for Hwaro, inspired by [Hugo Book](https://github.com/alex-shpak/hugo-book).

## Features

- **Sidebar Navigation** — Organized left sidebar with nested sections
- **Clean Typography** — Focused on reading comfort
- **Search** — Client-side full-text search
- **Shortcodes** — Hints, alerts, and tabs for rich content
- **Responsive** — Works on mobile with collapsible sidebar
- **Lightweight** — No JavaScript frameworks, no build step

## Quick Start

Get started by exploring the sidebar sections, or jump straight to the [Getting Started](/docs/getting-started/) guide.
