Even is a clean, responsive theme based on the Zola [Even theme](https://github.com/getzola/even) by olOwOlo, adapted for [Hwaro](https://github.com/hahwul/hwaro).

## Features

- Clean and minimalist design
- Responsive layout with mobile navigation
- Categories and tags support
- Pagination
- Syntax highlighting
- SEO optimized

## Quick Start

1. Install Hwaro
2. Create a new site: `hwaro init my-site`
3. Add the Even theme files
4. Run `hwaro serve` to preview

> Even focuses on readability and simplicity, making it perfect for a personal blog.

## Code Example

Here's a simple example of how to configure your site:

```toml
title = "My Blog"
description = "A personal blog powered by Hwaro"

[pagination]
enabled = true
per_page = 5
```

Enjoy writing with Even!
