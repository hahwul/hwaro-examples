This article offers a sample of basic Markdown syntax that can be used in Hwaro content files.

## Headings

The following HTML `<h1>`—`<h6>` elements represent six levels of section headings. `<h1>` is the highest section level while `<h6>` is the lowest.

## Paragraph

Xerum, currentibus iam nostrorum sententiis, nihil impedit quo minus id quod maxime placeat facere possimus, omnis voluptas assumenda est, omnis dolor repellendus. Temporibus autem quibusdam et aut officiis debitis aut rerum necessitatibus saepe eveniet ut et voluptates repudiandae sint et molestiae non recusandae.

## Blockquotes

The blockquote element represents content that is quoted from another source, optionally with a citation.

> Tiam, ad mint andae voluptua. Non provident, currentibus iam nostrorum sententiis, similique sunt in culpa qui officia deserunt mollitia animi, id est laborum et dolorum fuga.

## Tables

Tables aren't part of the core Markdown spec, but Hwaro supports them out of the box.

| Name  | Age |
|-------|-----|
| Bob   | 27  |
| Alice | 23  |

## Code Blocks

```html
<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>Example HTML5 Document</title>
</head>
<body>
  <p>Test</p>
</body>
</html>
```

## List Types

### Ordered List

1. First item
2. Second item
3. Third item

### Unordered List

- List item
- Another item
- And another item
