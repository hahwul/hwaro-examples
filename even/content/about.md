## About Even

Even is a clean, responsive theme adapted for [Hwaro](https://github.com/hahwul/hwaro), based on the [Even Hugo theme](https://github.com/olOwOlo/hugo-theme-even) by olOwOlo and the [Even Zola theme](https://github.com/getzola/even).

This demo site was created to showcase the Even theme's features and design.

## Features

- Clean and minimalist design
- Responsive layout with mobile navigation
- Categories and tags support
- Pagination
- Syntax highlighting
- SEO optimized with OpenGraph and Twitter Cards
- RSS feed generation
