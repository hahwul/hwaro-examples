Experience a next-generation static site generated by [Hwaro](https://github.com/hahwul/hwaro).

This template features an immersive ambient background with fluid glowing orbs, frosted glass components (`backdrop-filter`), and a dark futuristic aesthetic.

## Features

- Deep void backgrounds with glowing neon accents
- Immersive ambient background animations
- Translucent frosted glass elements (`backdrop-filter`)
- Sleek typography with **Space Grotesk** and **Inter**

## Quick Start

1. Edit `content/index.md` to change this page.
2. Add new `.md` files in `content/` to create new pages.
3. Run `hwaro build` to regenerate the site.
4. Run `hwaro serve` to preview changes locally.
