# About Acme Wiki

Acme Wiki is your team's central knowledge base. It serves as the single source of truth for documentation, guides, and reference materials across the organization.

## Purpose

This wiki provides structured, searchable documentation for all team members. Whether you're onboarding a new engineer or looking up a specific API reference, everything you need is organized and accessible.

## Contributing

All team members are encouraged to contribute. Documentation is written in Markdown and organized into logical categories. See the [Content Management](docs/guides/content-management/) guide for details on how to add or edit articles.

## Contact

For questions about this wiki, reach out to the documentation team or open an issue in the repository.
