This is a demo site for the **no-style-please** theme, a (nearly) no-CSS, fast, minimalist theme for [Hwaro](https://github.com/hahwul/hwaro).

The theme is inspired by [riggraz/no-style-please](https://github.com/riggraz/no-style-please), a minimalist Jekyll theme that uses almost no CSS to deliver a fast, readable experience.

## Features

- Monospace font for a clean, minimal look
- Dark mode support (auto, light, dark)
- ~1KB of CSS total
- Fast page loads with no external dependencies
- Simple navigation and post listing
