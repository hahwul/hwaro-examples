This template showcases how to leverage modern CSS features—like `backdrop-filter` and advanced gradient animations—to create a visually striking but lightweight website.

### The aesthetic

The background animation runs entirely on CSS, moving 5 distinct radial gradients in different directions with varying timing to create an organic, ever-shifting flow of colors that resembles silk.

The content panels use a subtle glassmorphism effect to remain legible while letting the vivid colors shine through.

### Features
* Modern glassmorphism UI
* Fluid, animated radial gradients
* Deep slate background (#07050f)
* Elegant typography (Inter & Outfit)
* No external dependencies or JavaScript for animations
