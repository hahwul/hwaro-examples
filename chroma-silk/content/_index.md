## Discover the flow

Welcome to **Chroma Silk**, an elegant portfolio template designed with fluid, animated background gradients and frosted glass elements. The aesthetic draws inspiration from the movement of silk under vivid lights.

Features:
- Animated CSS background gradients blending with `mix-blend-mode: screen`.
- Frosted glass containers utilizing `backdrop-filter: blur()`.
- Responsive layout with elegant hover animations.
- Beautiful typography featuring Outfit and Inter.
