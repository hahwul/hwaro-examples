Welcome to the Poison theme! This is the first post on this demo site.

## Getting Started

The Poison theme provides a sidebar navigation layout with a clean dark color scheme. It's designed for blogs that value readability and simplicity.

### Key Features

- Sidebar with hierarchical navigation links
- Dot-leader post listings for a classic look
- Table of contents for long articles
- Mobile-friendly responsive design

## Why Poison?

The original [Hugo Poison](https://github.com/lukeorth/poison) theme is known for its clean aesthetic and functional sidebar. This Hwaro port captures the same spirit with a pure CSS implementation.
