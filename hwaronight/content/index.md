# Welcome to HwaroNight

A dark theme for [Hwaro](https://github.com/hahwul/hwaro) inspired by the **Tokyo Night** color palette.

## Features

- Dark background with carefully chosen contrast
- Monospace typography using Source Code Pro
- Markdown-style heading prefixes
- Syntax highlighted code blocks
- Tag-based content organization
- Clean, minimal layout at 1024px max-width

## Quick Start

```bash
hwaro init mysite
cd mysite
# copy hwaronight templates and config
hwaro serve
```
