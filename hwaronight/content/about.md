# About HwaroNight

HwaroNight is a dark theme for the [Hwaro](https://github.com/hahwul/hwaro) static site generator, inspired by the popular **Tokyo Night** color scheme.

## Color Palette

The theme uses the Tokyo Night color palette:

| Color     | Hex       | Usage              |
|-----------|-----------|-------------------|
| Background | `#1a1b26` | Page background   |
| Foreground | `#a9b1d6` | Body text         |
| Cyan      | `#7dcfff` | Headings, accents |
| Orange    | `#ff9e64` | Links             |
| Green     | `#9ece6a` | Tags, code        |
| Magenta   | `#bb9af7` | Emphasis          |
| Yellow    | `#e0af68` | Dates, highlights |
| Red       | `#f7768e` | Errors, alerts    |
| Blue      | `#7aa2f7` | Selections        |

## Typography

The theme uses **Source Code Pro** as its primary font, giving the entire site a monospace aesthetic that pairs well with the dark color scheme.
