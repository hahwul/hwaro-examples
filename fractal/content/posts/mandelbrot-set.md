## z = z^2 + c

The Mandelbrot set is perhaps the most recognizable object in all of mathematics. Defined as the set of complex numbers c for which the iteration z = z^2 + c does not diverge when started from z = 0, its boundary encodes infinite complexity within a compact region of the complex plane.

## The Iteration

For each point c in the complex plane:

1. Begin with z = 0
2. Compute z = z^2 + c repeatedly
3. If |z| remains bounded (never exceeds 2), the point c belongs to the Mandelbrot set
4. If |z| escapes to infinity, the number of iterations before escape determines the color

This simple process -- repeated millions of times across a grid of points -- produces the iconic cardioid and circular bulbs connected by filaments of extraordinary delicacy.

## Boundary Complexity

The boundary of the Mandelbrot set has Hausdorff dimension 2, meaning it is so convoluted that it effectively fills a two-dimensional area despite being a one-dimensional curve. Zoom into any boundary region and new structures emerge: miniature copies of the whole set, spiral arms, dendrites, and seahorse valleys.

## Deep Zoom

At magnification factors exceeding 10^200, the set continues to reveal new structure. No two regions are identical, yet every region contains echoes of the whole. This is self-similarity -- not exact, but statistical -- the hallmark of fractal geometry.

## Coloring

The art lies in the mapping from iteration count to color. Smooth coloring algorithms interpolate between iteration bands, producing continuous color fields that reveal the dynamics of escape. Histogram equalization, orbit traps, and distance estimation each offer distinct visual interpretations of the same underlying mathematics.
