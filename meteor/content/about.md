# The Story of Meteor

Meteor is a demonstration site for [Hwaro](https://github.com/hahwul/hwaro), showcasing how to create an interactive, atmospheric experience using minimal tools and high-performance static generation.

## Design Philosophy

This site follows a "Graceful Dark" aesthetic:
- **No Gradients**: Every color transition is handled through opacity or solid color layering, following strict design constraints.
- **No Emojis**: Focuses on typography and animation to convey emotion and atmosphere.
- **High Performance**: The background animation is powered by a lightweight Canvas engine that respects system resources.

## Built with Hwaro

Hwaro is a fast static site generator written in Crystal. It allows for:
- **Rapid Prototyping**: Instant build times.
- **Flexible Templating**: Using Jinja2-style templates.
- **Minimal Footprint**: No heavy runtime dependencies.
