# Welcome to Aurora Drift

Experience the ethereal beauty of a cosmic aurora combined with modern **glassmorphism** design. This theme utilizes dynamic, drifting gradients in cyan, magenta, and green, layered underneath translucent frosted glass panels.

## Key Features

- **Drifting Orbs**: Animated glowing orbs that drift slowly in the background, simulating polar lights.
- **Frosted Glass UI**: Clean, translucent containers using `backdrop-filter` for a modern aesthetic.
- **Modern Typography**: A beautiful pairing of *Space Grotesk* for headings and *Inter* for readable body text.
- **Deep Void Background**: A carefully chosen deep space color palette to make the glowing elements pop.

## Explore the Cosmos

1. Navigate to the [About page](/about/) to see another example of the glass panel.
2. Check out the styling in `templates/header.html` to customize the animation speeds and colors.

```css
/* Example customization */

}
```
