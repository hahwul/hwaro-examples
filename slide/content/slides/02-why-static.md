# Why Static Sites?

Static sites solve real problems that dynamic sites create.

## Performance

Pre-built HTML means **zero server-side rendering**. Every page loads instantly from a CDN edge node.

## Security

No database, no server runtime, no attack surface. Static files are inherently secure.

## Reliability

No server to crash, no process to restart. If your CDN is up, your site is up.

## Cost

Hosting static files is essentially **free** on platforms like GitHub Pages, Netlify, or Cloudflare Pages.
