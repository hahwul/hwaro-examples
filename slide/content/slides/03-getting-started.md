# Getting Started

Install Hwaro and scaffold a new project in seconds.

## Installation

```bash
brew install hwaro
```

## Create a Project

```bash
hwaro init my-site
cd my-site
```

## Project Structure

```
my-site/
  config.toml
  content/
    index.md
  templates/
    header.html
    footer.html
    page.html
    section.html
```

Every project starts with a **config**, a **content** directory, and **templates**.
