## About Me

Hi there! I'm a developer and blogger who loves building things with code. This site is built with [Hwaro](https://github.com/hahwul/hwaro), a fast and lightweight static site generator written in Crystal.

## About This Site

This is a demonstration of the **Hermit** theme for Hwaro, inspired by the [hermit_zola](https://github.com/VersBinarii/hermit_zola) theme. It features a minimal dark design focused on readability and performance.

### Features

- Dark color scheme for comfortable reading
- Responsive design for all devices
- Syntax highlighting for code blocks
- RSS feed support
- Tag-based content organization
- Fast page loads with minimal JavaScript
