## About the event

The Podium Rush World Championship is a single-day track festival built around the 100m sprint final. Eight lanes. Eight stories. One podium. Each heat is streamed live with laser-line finish verification; every record is ratified by three independent timing systems before results are declared official.

This edition marked the fastest final on record, with the first five athletes all dipping under 9.85 seconds and two new continental records set in the semi-finals earlier the same afternoon.
