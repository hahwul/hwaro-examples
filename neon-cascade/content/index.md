Welcome to **Neon Cascade**. This Hwaro example site showcases a dark-themed aesthetic combining deep space elements with cyberpunk neon accents and translucent frosted glass panels (`backdrop-filter: blur()`).

This theme is designed for modern blogs, portfolios, or documentation sites that want to stand out with a bold, atmospheric design.

## Features

- **Glassmorphism UI**: Beautiful, translucent panels that blur the animated background behind them.
- **Neon Accents**: Vibrant cyan, magenta, and purple glows to guide the eye.
- **Animated Background**: Floating orbs that create a dynamic depth effect.
- **Responsive Design**: Scales elegantly from desktop to mobile.
