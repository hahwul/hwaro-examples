# About Murano Glass
Murano Glass is a luminous theme inspired by the art of Venetian glassblowing with translucent colors.
## Features
- Responsive layout for all devices
- Custom color palette and typography
- Clean post cards with hover effects
- Tag-based content organization
- RSS feed support
