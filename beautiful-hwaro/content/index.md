## Welcome!

This is a sample website built with [Hwaro](https://github.com/hahwul/hwaro), inspired by the [Beautiful Jekyll](https://beautifuljekyll.com) theme by Dean Attali.

Beautiful Hwaro demonstrates how to create a modern, clean blog with Hwaro's fast static site generation capabilities. It features:

- **Clean typography** with Lora and Open Sans fonts
- **Responsive design** that looks great on all devices
- **Blog section** with post listings, tags, and pagination
- **Syntax highlighting** for code blocks
- **SEO optimized** with OpenGraph and Twitter Cards
