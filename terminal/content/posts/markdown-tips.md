Hwaro uses Markdown for content. Here are some tips.

## Text formatting

- **Bold** with `**bold**`
- *Italic* with `*italic*`
- `Inline code` with backticks

## Code blocks

Use triple backticks with a language identifier:

```crystal
puts "Hello from Crystal!"
```
