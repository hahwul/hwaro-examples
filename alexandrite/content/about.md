## About Alexandrite

This is a glamorous, color-shifting blog theme for [Hwaro](https://github.com/hahwul/hwaro), the fast static site generator written in Crystal.

Inspired by the alexandrite gemstone — which shifts from emerald green in daylight to ruby red under incandescent light — this theme embraces bold visual design with vibrant gradients, smooth animations, and rich color palettes.

### Features

- **Color-shifting gradients** that transition between emerald and ruby
- **Smooth CSS animations** for dynamic visual impact
- **Responsive design** that works across all devices
- **Dark theme** with vivid accent colors
- **Clean typography** that balances glamour with readability

### Built With

- [Hwaro](https://github.com/hahwul/hwaro) — Fast static site generator
- Pure CSS animations and effects — no JavaScript frameworks required
- [Google Fonts](https://fonts.google.com/) — Playfair Display & Inter
