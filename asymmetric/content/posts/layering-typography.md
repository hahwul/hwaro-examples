+++
title = "Layering in Web Typography"
date = "2024-03-05"
summary = "How overlapping text and images creates depth without using shadows."
categories = ["Typography"]
+++

Historically, we've relied on box-shadows to create depth on the web. However, there is a more structural way to achieve it: layering. By allowing text blocks to physically intersect with images or decorative background shapes, we create a sense of foreground and background organically.

This approach requires careful attention to contrast to ensure text remains legible, but when executed correctly, it produces a dynamic, editorial feel that standard single-column layouts struggle to achieve.
